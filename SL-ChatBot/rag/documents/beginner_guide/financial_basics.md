# 금융 기초 개념 📚

## 개념

투자를 시작하기 전에 꼭 알아야 할 **금융의 기본 개념과 용어**들을 쉽게 설명합니다.
재무제표를 읽고 기업을 분석하는 데 필요한 핵심 지식을 담았습니다.

> 💡 **핵심 아이디어**: 숫자로 기업을 읽는 법을 배워봅시다!

---

## 1. 주식이란? 📈

### 기본 개념
**주식(Stock)**은 회사의 **소유권 일부**를 나타내는 증서입니다.
- 주식을 사면 = 그 회사의 주주(주인)가 됨
- 회사가 잘되면 = 주가 상승 + 배당금
- 회사가 못되면 = 주가 하락 + 손실

### 주식의 종류
| 종류 | 설명 | 특징 |
|------|------|------|
| **보통주** | 일반적인 주식 | 의결권 있음, 배당 변동 |
| **우선주** | 우선 배당 주식 | 의결권 없음, 배당 우선 |

## 2. 재무제표 이해하기 📊

재무제표는 **"기업의 성적표"**입니다. 크게 3가지로 구성됩니다.

### 📋 손익계산서 (Income Statement)
**"얼마나 벌었나?"** - 일정 기간 동안의 수익과 비용

```
매출액 (Revenue)           1,000억원
- 매출원가 (COGS)            600억원
= 매출총이익 (Gross Profit)   400억원
- 영업비용 (Operating Exp.)   200억원
= 영업이익 (Operating Income) 200억원
- 금융비용 (Interest)          20억원
= 당기순이익 (Net Income)     180억원
```

### 📋 재무상태표 (Balance Sheet)
**"무엇을 가지고 있나?"** - 특정 시점의 자산, 부채, 자본

```
[자산 = 부채 + 자본]

자산 (Assets)          1,000억원
├─ 유동자산              400억원 (현금, 재고 등)
└─ 비유동자산            600억원 (건물, 장비 등)

부채 (Liabilities)       400억원
├─ 유동부채              200억원 (1년 내 갚을 빚)
└─ 비유동부채            200억원 (장기 빚)

자본 (Equity)           600억원 (순자산)
```

### 📋 현금흐름표 (Cash Flow Statement)
**"현금이 어떻게 움직였나?"** - 현금의 유입과 유출

- **영업활동**: 본업에서 번 현금
- **투자활동**: 투자로 쓴 현금
- **재무활동**: 차입/상환한 현금

## 3. 핵심 금융 지표 💰

### 수익성 지표

#### ROE (자기자본이익률)
```
ROE = 당기순이익 / 자기자본 × 100
```
- **의미**: 주주 돈으로 얼마나 벌었나?
- **기준**: 10% 이상이면 양호
- **예시**: ROE 15% = 100원 투자해서 15원 벌었다

#### ROA (총자산이익률)
```
ROA = 당기순이익 / 총자산 × 100
```
- **의미**: 전체 자산으로 얼마나 벌었나?
- **기준**: 5% 이상이면 양호
- **예시**: ROA 8% = 자산 활용 효율이 좋다

### 가치평가 지표

#### PER (주가수익비율)
```
PER = 주가 / 주당순이익(EPS)
```
- **의미**: 1원 벌기 위해 몇 원을 투자?
- **기준**: 업종별로 다름 (10~20배 일반적)
- **예시**: PER 10배 = 10년이면 본전 회수

#### PBR (주가순자산비율)
```
PBR = 주가 / 주당순자산(BPS)
```
- **의미**: 회사 자산 대비 주가
- **기준**: 1 이하면 저평가 가능성
- **예시**: PBR 0.8 = 자산가치보다 싸게 거래

### 안정성 지표

#### 부채비율
```
부채비율 = 부채 / 자기자본 × 100
```
- **의미**: 빚이 얼마나 많은가?
- **기준**: 100% 이하 안정, 200% 이상 위험
- **예시**: 부채비율 50% = 자본보다 빚이 적다

#### 유동비율
```
유동비율 = 유동자산 / 유동부채 × 100
```
- **의미**: 단기 지급능력
- **기준**: 150% 이상 양호
- **예시**: 유동비율 200% = 단기 부채 갚을 여력 충분

## 4. 시장 용어 사전 📖

### 기본 용어
| 용어 | 뜻 | 설명 |
|------|-----|------|
| **시가총액** | 회사 전체 가치 | 주가 × 발행주식수 |
| **거래량** | 거래된 주식 수 | 유동성 지표 |
| **배당** | 이익 분배 | 주주에게 돌려주는 돈 |
| **증자** | 자본금 증가 | 새 주식 발행 |
| **감자** | 자본금 감소 | 주식 수 줄이기 |

### 투자 용어
| 용어 | 뜻 | 설명 |
|------|-----|------|
| **매수** | 사기 | 주식 구매 |
| **매도** | 팔기 | 주식 판매 |
| **호가** | 희망 가격 | 사고 싶은/팔고 싶은 가격 |
| **체결** | 거래 성사 | 매매 완료 |
| **포트폴리오** | 투자 조합 | 보유 종목들의 집합 |

## 5. 주식 거래 기초 💹

### 주문 방식
1. **시장가 주문**: 바로 거래 (현재 가격)
2. **지정가 주문**: 원하는 가격 지정
3. **조건부 주문**: 특정 조건 충족시 주문

### 거래 시간
- **정규장**: 오전 9:00 ~ 오후 3:30
- **시간외**:
  - 장전: 08:30 ~ 09:00
  - 장후: 15:40 ~ 16:00

---

## 실전 예시: 기업 분석하기 🔍

### 삼성전자를 예로 들면:

```
1. 수익성 체크
   - ROE: 15% (양호 ✅)
   - 영업이익률: 20% (우수 ✅)

2. 가치평가
   - PER: 12배 (업계 평균 15배보다 낮음)
   - PBR: 1.5배 (적정 수준)

3. 안정성
   - 부채비율: 30% (매우 안정 ✅)
   - 현금 보유: 충분

결론: 재무적으로 건전한 기업
```

## 초보자가 자주 하는 실수 ⚠️

### 1. 하나의 지표만 보기
❌ "PER이 낮으니 무조건 좋은 주식이다"
✅ 여러 지표를 종합적으로 판단

### 2. 절대값으로만 판단
❌ "ROE 5%는 무조건 나쁘다"
✅ 업종 특성과 경쟁사 비교 필요

### 3. 과거 데이터만 믿기
❌ "작년에 좋았으니 올해도 좋을 것"
✅ 미래 전망도 함께 고려

## 자주 묻는 질문 (FAQ) ❓

**Q: 어떤 지표를 먼저 봐야 하나요?**
A: PER → ROE → 부채비율 순으로 보시면 좋습니다. 가격 → 수익성 → 안정성 순서입니다.

**Q: 재무제표는 어디서 볼 수 있나요?**
A: 금융감독원 전자공시시스템(DART), 네이버 금융, 각 증권사 앱에서 확인 가능합니다.

**Q: 좋은 기업의 기준은 무엇인가요?**
A: ROE 10% 이상, 부채비율 100% 이하, 꾸준한 매출 성장이 기본입니다.

**Q: 모든 지표가 좋으면 무조건 좋은 투자인가요?**
A: 아닙니다. 미래 성장성, 산업 전망, 주가 수준 등도 함께 고려해야 합니다.
