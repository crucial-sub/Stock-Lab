# 투자 시작하기 🚀

## 개념

주식 투자를 **처음 시작하는 분들을 위한 실전 가이드**입니다.
계좌 개설부터 첫 투자까지, 단계별로 차근차근 설명합니다.

> 💡 **핵심 아이디어**: 작게 시작하고, 꾸준히 배우며, 천천히 성장하세요!

---

## 투자 시작 로드맵 🗺️

```
1단계: 준비하기 (1주)
  ├─ 투자 목표 설정
  ├─ 증권 계좌 개설
  └─ 기초 지식 학습

2단계: 연습하기 (2-4주)
  ├─ 모의투자 체험
  ├─ 관심종목 분석
  └─ 투자전략 수립

3단계: 실전투자 (지속)
  ├─ 소액 투자 시작
  ├─ 포트폴리오 구성
  └─ 정기 리밸런싱
```

---

## 1단계: 준비하기 📝

### 1-1. 투자 목표 설정

#### 목표 설정 체크리스트
- [ ] **투자 기간**: 단기(~1년) / 중기(1~3년) / 장기(3년~)
- [ ] **목표 수익률**: 연 5% / 10% / 15% 이상
- [ ] **투자 가능 금액**: 월 ___만원
- [ ] **리스크 허용도**: 보수적 / 중립 / 공격적

#### 목표 설정 예시
```
👤 30대 직장인 A씨
- 목표: 10년 후 주택 자금 마련
- 기간: 장기 (10년)
- 월 투자금: 100만원
- 목표 금액: 2억원
- 전략: 성장주 50% + 배당주 30% + 팩터투자 20%
```

### 1-2. 증권 계좌 개설

#### 증권사 선택 기준
| 기준 | 중요도 | 체크 사항 |
|------|--------|----------|
| **수수료** | ⭐⭐⭐⭐⭐ | 거래 수수료 0.015% 이하 |
| **플랫폼** | ⭐⭐⭐⭐ | 모바일 앱 사용성 |
| **정보 제공** | ⭐⭐⭐ | 리서치 자료, 투자 정보 |
| **해외투자** | ⭐⭐ | 해외 주식 거래 가능 여부 |

#### 계좌 개설 필요 서류
- 신분증
- 휴대폰 (본인 명의)
- 계좌 연결용 은행 계좌

### 1-3. 기초 지식 학습

#### 필수 학습 항목
1. **재무제표 읽기** (1주차)
   - 손익계산서, 재무상태표 이해
   - PER, ROE 등 기본 지표

2. **주문 방법** (2주차)
   - 시장가, 지정가 주문
   - 거래 시간, 호가 단위

3. **리스크 관리** (3주차)
   - 손절매, 분산 투자
   - 포트폴리오 이론 기초

---

## 2단계: 연습하기 🎮

### 2-1. 모의투자 활용

#### 모의투자 플랫폼
- 증권사 제공 모의투자
- 네이버 금융 모의투자
- 투자 대회 참가

#### 모의투자 실습 과정
```
1주차: 관심 종목 10개 선정
2주차: 매수/매도 연습
3주차: 손익 분석
4주차: 전략 수정
```

### 2-2. 종목 분석 실습

#### 초보자용 분석 템플릿
```markdown
## 기업명: [회사 이름]

### 1. 기본 정보
- 업종:
- 시가총액:
- 주요 사업:

### 2. 재무 분석
- PER: ___ (업종 평균: ___)
- ROE: ___%
- 부채비율: ___%
- 매출 성장률: ___%

### 3. 투자 포인트
- 강점:
- 약점:
- 기회:
- 위협:

### 4. 투자 결정
- 매수/관망/제외
- 목표 가격:
- 손절 가격:
```

### 2-3. 나만의 투자 원칙 만들기

#### 투자 원칙 예시
```
📋 나의 투자 원칙

1. 종목당 최대 10% 이하 투자
2. 손실 -10% 도달시 무조건 손절
3. 월 1회 포트폴리오 점검
4. 뉴스에 휘둘리지 않기
5. 이해하지 못하는 기업 투자 금지
```

## 3단계: 실전 투자 💼

### 3-1. 첫 투자 시작하기

#### 초보자 추천 첫 투자 방법

##### 방법 1: ETF로 시작
```
✅ 장점: 자동 분산투자, 낮은 리스크
✅ 추천 ETF 종류:
  - KOSPI200 ETF: 대형주 투자
  - 고배당 ETF: 배당 수익
  - 팩터 ETF: 팩터 전략
```

##### 방법 2: 우량주 소액 투자
```
✅ 장점: 안정적, 학습 효과
✅ 선정 기준:
  - 시가총액 상위 30위
  - 10년 이상 영업이익 흑자
  - 배당 지속 기업
```

##### 방법 3: 팩터 투자
```
✅ 장점: 체계적, 감정 배제
✅ 시작 방법:
  - 단순 팩터 선택 (예: PER)
  - 상위 20개 종목 선정
  - 동일 가중 투자
```

### 3-2. 포트폴리오 구성

#### 초보자 포트폴리오 예시

##### 안전형 (보수적)
```
현금 30%
├─ 비상시 추가 매수용
│
주식 70%
├─ 대형 우량주: 40%
├─ 배당주: 20%
└─ ETF: 10%
```

##### 균형형 (중립적)
```
현금 20%
├─ 기회 포착용
│
주식 80%
├─ 가치주: 30%
├─ 성장주: 30%
├─ 팩터 전략: 20%
```

##### 성장형 (공격적)
```
현금 10%
├─ 최소 현금
│
주식 90%
├─ 성장주: 40%
├─ 중소형주: 30%
├─ 테마주: 20%
```

### 3-3. 리스크 관리

#### 필수 리스크 관리 규칙

##### 1. 분산 투자
```
최소 규칙:
- 종목 수: 10개 이상
- 종목당 비중: 최대 15%
- 업종 분산: 3개 이상
```

##### 2. 손절매 규칙
```
-7% 규칙:
매수가 대비 -7% 하락시 매도

-10% 규칙:
어떤 경우든 -10% 도달시 매도
```

##### 3. 투자 일지 작성
```markdown
## 2025-01-15 투자 일지

### 매수
- 종목: A기업
- 이유: PER 8배, ROE 15%
- 금액: 100만원
- 목표: +20%

### 매도
- 종목: B기업
- 이유: 목표 달성
- 수익률: +15%

### 반성
- 좋았던 점:
- 개선할 점:
```

## 투자 심리 관리 🧠

### 피해야 할 심리적 함정

#### 1. FOMO (Fear Of Missing Out)
```
증상: "다들 사는데 나만 안 사면..."
해결: 투자 원칙 고수, 충동 매수 금지
```

#### 2. 손실 회피
```
증상: "조금만 더 기다리면 오를 거야..."
해결: 손절매 규칙 철저히 준수
```

#### 3. 확증 편향
```
증상: "내가 산 주식은 무조건 오른다"
해결: 객관적 지표로만 판단
```

#### 4. 과신
```
증상: "몇 번 성공했으니 이제 다 안다"
해결: 겸손한 자세, 지속적 학습
```

## 자주 묻는 질문 (FAQ) ❓

**Q: 얼마로 시작하는 게 좋나요?**
A: 월 여유자금의 30~50% 이하로 시작하세요. 최소 100만원 이상이면 분산투자가 가능합니다.

**Q: 첫 종목은 어떻게 고르나요?**
A: 자신이 잘 아는 기업이나 대형 우량주부터 시작하세요. ETF도 좋은 선택입니다.

**Q: 하루에 얼마나 시간을 투자해야 하나요?**
A: 장기투자라면 주 1~2시간이면 충분합니다. 단기 매매는 매일 1시간 이상 필요합니다.

**Q: 손실이 나면 어떻게 하나요?**
A: 미리 정한 손절 규칙을 지키세요. 감정적 대응은 더 큰 손실로 이어집니다.

**Q: 언제 팔아야 하나요?**
A: 목표 수익률 달성, 투자 논리 변경, 더 좋은 기회 발견시 매도를 고려하세요.

## 초보자가 꼭 피해야 할 것들 ⚠️

### ❌ 절대 하지 마세요
1. **신용/미수 거래**: 빚내서 투자
2. **한 종목 올인**: 전 재산 한 종목 투자
3. **소문 투자**: 검증 없는 정보로 투자
4. **추격 매수**: 급등 후 뒤늦게 매수
5. **나락 칼 받기**: 급락 중 무작정 매수

### ⚠️ 주의하세요
1. **테마주**: 변동성 크고 위험
2. **작전주**: 조작 가능성
3. **신규 상장주**: 가격 불안정
4. **레버리지 상품**: 위험도 2배
5. **해외 펜스톡**: 정보 부족
