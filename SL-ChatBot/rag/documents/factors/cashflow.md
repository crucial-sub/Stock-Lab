# 💵 **CASHFLOW 팩터 (8개) — 초보자용 완전 정리**

현금흐름 팩터는
**기업이 실제로 돈을 얼마나 잘 벌고 있는지, 어디에 쓰고 있는지, 남는 돈이 얼마나 있는지**를 평가한다.

> 💡 핵심 포인트:
> “회계상 이익”이 아니라 **진짜 현금 흐름**을 본다.

---

# 🔵 **1) CFO_TO_SALES (영업현금흐름 / 매출액)**

### ✔ 정의

매출액 중 실제로 현금이 얼마나 들어오는지를 측정.

### ✔ 공식

```
CFO / 매출액 × 100
```

### ✔ 해석

* 10%↑ → 현금창출력 우수
* 5% 미만 → 현금화 능력 약함

### ✔ 예시

매출 1조, CFO 1200억 → 12%
→ 매우 우수한 수준

### ✔ 주의

* 재고 증가·매출채권 증가로 CFO가 일시적으로 훼손될 수 있음

---

# 🔵 **2) CFI_TO_ASSETS (투자현금흐름 / 자산)**

### ✔ 정의

회사가 자산 대비 얼마나 투자를 하고 있는지.

### ✔ 공식

```
투자CF / 총자산
```

### ✔ 해석

* 값이 **음수** → 정상(대부분 투자는 현금 유출)
* 너무 큰 음수 → 과도한 투자 부담
* 양수 → 보유 자산 매각 중(위험 신호일 수도 있음)

### ✔ 예시

총자산 10조, 투자CF -5,000억 → -5%
→ 정상적 투자 범위

### ✔ 주의

* 성장 기업은 투자CF가 항상 음수
* 지속적 양수는 “자산 매각으로 버티는 회사” 가능성

---

# 🔵 **3) CFF_RATIO (재무현금흐름 비율)**

### ✔ 정의

차입·상환·배당·자사주 등 **재무 활동**으로 발생한 현금 흐름.

### ✔ 공식

```
CFF / 총자산
```

### ✔ 해석

* 양수 → 차입금 증가 or 자본 조달
* 음수 → 차입 상환 or 배당 지급

### ✔ 예시

재무CF +3,000억 → 차입 증가
재무CF -2,000억 → 이자·상환·배당 지급

### ✔ 주의

* 단독 해석 어려움 → CFO·CFI와 조합해야 의미 있음

---

# 🔵 **4) FREE_CASH_FLOW (잉여현금흐름, FCF)**

### ✔ 정의

기업이 본업으로 벌어들인 현금에서
**유지·투자(CAPEX) 지출을 뺀 후 남는 돈**.

### ✔ 공식

```
FCF = 영업현금흐름 - CAPEX
```

### ✔ 해석

* 양수 → 투자해도 돈이 남음 (건전한 기업)
* 음수 → 투자비가 본업 현금을 초과
  (성장기 기업이거나, 위험 신호일 수도 있음)

### ✔ 예시

CFO 5,000억 – CAPEX 3,000억 = 2,000억
→ 잉여현금 + (긍정적)

### ✔ 주의

성장 산업은 FCF가 여러 해 음수여도 정상.

---

# 🔵 **5) CAPEX_TO_SALES (자본지출 / 매출)**

### ✔ 정의

매출 대비 설비·투자(CAPEX)에 얼마나 쓰는지.

### ✔ 공식

```
CAPEX / 매출액 × 100
```

### ✔ 해석

* 높음 → 설비투자 활발 (성장 가능성 ↑, 비용 부담 ↑)
* 낮음 → 성숙기 기업, 안정적

### ✔ 예시

CAPEX 5,000억 / 매출 2조 = 25%
→ 설비투자 매우 활발

### ✔ 주의

* 반도체·통신·플랜트 등 CAPEX 산업은 원래 높음

---

# 🔵 **6) CAPEX_TO_OCF (자본지출 / 영업현금흐름)**

### ✔ 정의

본업으로 벌어들인 현금 중 **얼마나 투자를 위해 쓰는지**.

### ✔ 공식

```
CAPEX / CFO × 100
```

### ✔ 해석

* 100%↑ → 벌어들인 돈보다 투자를 더 많이 함
* 30~70% → 건강한 투자 규모
* 너무 낮으면 → 성장성 낮음

### ✔ 예시

CFO 4,000억 / CAPEX 3,000억 → 75%
→ 성장 위한 투자 활발

### ✔ 주의

경기민감업종은 CAPEX가 변동 심함.

---

# 🔵 **7) FCF_TO_SALES (잉여현금흐름 / 매출액)**

### ✔ 정의

매출 대비 얼마의 **순수 현금이 남는지** 평가하는 지표.

### ✔ 공식

```
FCF / 매출액 × 100
```

### ✔ 해석

* 높을수록 → 훌륭한 현금창출력
* 낮을수록 → 본업 수익성이 약함

### ✔ 예시

매출 1조, FCF 1,500억 → 15%
→ 매우 우수

### ✔ 주의

* 단기 CAPEX 변동 때문에 흔들릴 수 있음

---

# 🔵 **8) FCF_MARGIN (FCF 마진)**

### ✔ 정의

순이익 마진과 비슷하지만
**회계 이익이 아닌 실제 현금 기반의 수익성**을 보여줌.

### ✔ 공식

```
FCF 마진 = (FCF / 매출액) × 100
```

### ✔ 해석

* 10%↑ → 매우 우수
* 5% 근처 → 안정
* 0% 미만 → 재무구조 약함

### ✔ 예시

매출 2조, FCF 3,000억 → 15%
→ 현금 기반 수익성 매우 강함

### ✔ 주의

CAPEX 폭증 시 일시적으로 급락 가능

---

# 🎯 **CASHFLOW 팩터 요약표**

| 팩터          | 의미            | 핵심 포인트       |
| ----------- | ------------- | ------------ |
| CFO/Sales   | 현금 기반 매출 효율   | 돈이 진짜 들어오나?  |
| CFI/Assets  | 투자 강도         | 성장 vs 위험     |
| CFF Ratio   | 차입/상환/배당 흐름   | 재무활동 분석      |
| FCF         | 남는 돈          | 가장 중요한 재무 지표 |
| CAPEX/Sales | 성장 투자 강도      | 산업별 차이 큼     |
| CAPEX/OCF   | 본업 현금 중 투자 비중 | 성장성 판단       |
| FCF/Sales   | 매출 대비 잉여현금    | 현금수익성        |
| FCF Margin  | 현금 기반 마진      | 기업 질 판단      |

---

# 🎯 **최종 결론**

CASHFLOW 팩터는
회계상 숫자가 아닌 **기업의 진짜 체력**을 보여준다.

가장 중요한 핵심 3개는:

* **FCF (Free Cash Flow)** — 남는 돈
* **CFO_TO_SALES** — 본업이 돈을 얼마나 벌어주는지
* **FCF_MARGIN** — 현금 기반 수익성

현금흐름이 좋은 기업은
경기침체에서도 버티고,
장기적으로 주가도 안정적으로 상승하는 특성이 있다.
