
# 🧩 **COMPOSITE 팩터 (3개)**

Composite(복합) 팩터는 여러 재무 지표를 합쳐
**기업의 질 / 수익률 / 주주환원 정도를 종합적으로 평가하는 지표**다.

---

# 🔵 **1) PIOTROSKI_F_SCORE (피오트로스키 F-Score)**

### ✔ 정의

9개 재무지표를 활용해 기업의 **재무건전성·수익성·안정성**을 종합 평가한 점수(0~9점).

### ✔ 구성 지표

1. 순이익 증가
2. 영업현금흐름 증가
3. ROA 증가
4. 영업현금흐름 > 순이익
5. 부채비율 감소
6. 유동성 증가
7. 발행여부(주식발행 없음)
8. 매출총이익 증가
9. 자산회전율 증가

### ✔ 공식

```
F-score = 위 9개 항목 각각 1점 또는 0점 → 총합 (0~9)
```

### ✔ 해석

* 7~9 → 매우 우량
* 4~6 → 보통
* 0~3 → 매우 위험

### ✔ 예시

7점 → 재무구조 매우 건강
3점 → 구조적 악화 가능성

### ✔ 주의

* 제조업 중심 개발 → 산업별 해석 다름
* 단기 급등/악재 영향 적음

---

# 🔵 **2) ENTERPRISE_YIELD (기업수익률)**

### ✔ 정의

EV(기업가치) 대비 벌어들이는 이익(EBIT)이 얼마나 되는지.

→ “이 회사가 실제 가치 대비 얼마나 ‘싼가’를 평가”

### ✔ 공식

```
Enterprise Yield = (EBIT / EV) × 100
```

### ✔ 해석

* 높을수록 → 저평가
* 낮을수록 → 고평가

### ✔ 예시

EBIT 1조, EV 10조 → 10%
→ 매우 매력적인 수익률

### ✔ 주의

* EV는 부채 포함 → 레버리지 기업은 왜곡 가능
* 세금·이자 제외 EBIT 기준이라 비교 안정적

---

# 🔵 **3) SHAREHOLDER_YIELD (주주수익률)**

### ✔ 정의

주주에게 돌아가는 **전체 수익률**을 평가하는 지표.

```
주주수익률 = 배당수익률 + 자사주매입수익률
```

### ✔ 해석

* 높을수록 → 주주 친화적 기업
* 배당+자사주 모두 중요
* 성장성 낮은 기업도 주주환원 강하면 점수 높음

### ✔ 예시

* 배당수익률 3%
* 자사주 매입수익률 2%
  → Shareholder Yield = 5%

### ✔ 주의

* 자사주 매입 후 실제 소각 여부 확인 필요
* 배당은 정책 변화 빨리 반영됨

---

# 🎯 **COMPOSITE 팩터 요약**

| 팩터                    | 의미        | 핵심 포인트   |
| --------------------- | --------- | -------- |
| **F-Score**           | 재무체질 종합점수 | 7점↑ 우량   |
| **Enterprise Yield**  | EV 대비 이익률 | 높을수록 저평가 |
| **Shareholder Yield** | 주주환원 종합   | 배당 + 자사주 |

---

# 🎯 **최종 결론**

COMPOSITE 팩터는 **단일 지표보다 더 신뢰도 높은 판단 기준**이다.

특히:

* F-Score → 재무 품질 전체 확인
* Enterprise Yield → 기업가치 대비 수익성
* Shareholder Yield → 얼마나 주주 친화적인가

이 3개는 단독으로도 강력하지만
**가치(Value), 퀄리티(Quality), 성장(Growth)** 과 함께 조합하면
퀀트 투자에서 가장 좋은 성능을 보이는 핵심 팩터들이다.
