# 복합 팩터 (Composite Factor)

📌 개념

복합 팩터는 **"여러 개의 단일 팩터를 조합해서 포괄적인 투자 신호를 만드는"** 전략이에요.
VALUE, QUALITY, GROWTH 등 여러 팩터를 함께 고려해서 더 정확한 투자 판단을 합니다.

💡 핵심: "여러 신호를 동시에 본다면?"

# 복합 팩터 주요 전략

## 1. 멀티팩터 스코어 (Multifactor Score)
✔ 정의

여러 팩터의 점수를 통합해서 종합 등급 만들기

✔ 방식

VALUE 점수 + QUALITY 점수 + GROWTH 점수 = 종합점수
또는 가중평균: (VALUE×30%) + (QUALITY×40%) + (GROWTH×30%)

✔ 해석

높은 점수 → 여러 조건 동시 만족
낮은 점수 → 조건 부족

✔ 장점

편향된 투자 피하고 균형잡힌 종목 선별

## 2. 그레이엄 지수 (Graham Index / Magic Formula)
✔ 정의

Joel Greenblatt의 "마법공식": 품질×가치 조합

✔ 방식

좋은 회사(높은 수익률) + 싼 주가(낮은 PEG) 결합

✔ 해석

마법공식 순위 높음 → 매력적
역사적으로 장기 아웃퍼폼

✔ 주의

시장 사이클에 따라 성과 변동

## 3. Piotroski F-Score
✔ 정의

财무 건전성을 평가하는 9개 지표 점수 (0~9점)

✔ 평가항목

1. 영업현금흐름 > 0
2. 순이익 > 0
3. ROA 증가
4. FCF > 순이익
5. 부채비율 감소
6. 유동비율 증가
7. 자산증가 낮음
8. 총이익률(Gross Margin) 높음
9. 자산회전율 높음

✔ 해석

8~9점 → 매우 우수
5~7점 → 중간
0~4점 → 부실 가능성

## 4. Altman Z-Score (재무위기 진단)
✔ 정의

기업의 부도 위험도를 평가하는 종합 지표

✔ 공식

Z = 1.2X₁ + 1.4X₂ + 3.3X₃ + 0.6X₄ + 1.0X₅

(X₁~X₅ = 유동성, 이익률, EBIT, 자기자본비율, 자산회전율)

✔ 해석

Z > 3 → 건전
1.8 < Z < 3 → 회색지대
Z < 1.8 → 부도 위험

## 5. PEG × 자산회전율 (효율성 복합평가)
✔ 정의

성장성(PEG)과 자산효율성 동시 평가

✔ 해석

PEG 낮음 + 자산회전율 높음 → 최고 등급
반대면 매력 떨어짐

## 6. 모멘텀 + 가치 (Momentum & Value Blend)
✔ 정의

최근 상승세(모멘텀) + 저평가(가치) 동시 만족

✔ 해석

가치주만 사면 주가 안 올라가고
모멘텀주만 사면 과열 위험

두 가지 동시 만족 기업이 최고 성과

## 7. 채권 금리 대비 주식수익률 (Equity Risk Premium)
✔ 정의

국채 금리 대비 주식의 수익률이 매력적인지 평가

✔ 공식

Earnings Yield / 채권금리 > 1.5 시 매력적

✔ 해석

금리가 낮으면 주식이 더 매력적
금리가 높으면 주식 매력도 낮아짐

# 🎯 복합 팩터 활용 예시

### 예시 1: 밸런스드 포트폴리오
- 가치(VALUE) 40%
- 품질(QUALITY) 30%
- 성장(GROWTH) 20%
- 기술(TECHNICAL) 10%

### 예시 2: 보수적 포트폴리오
- 가치(VALUE) 50%
- 품질(QUALITY) 40%
- 현금흐름(CASHFLOW) 10%

### 예시 3: 공격적 포트폴리오
- 성장(GROWTH) 50%
- 모멘텀(MOMENTUM) 30%
- 효율성(EFFICIENCY) 20%

# 🎯 복합 팩터 요약표

| 전략 | 의미 | 사용처 |
|------|------|--------|
| Multifactor Score | 여러 팩터 통합 점수 | 종합 평가 |
| Magic Formula | 품질 + 가치 조합 | 장기 투자 |
| F-Score | 재무 건전성 진단 | 위험 회피 |
| Z-Score | 부도 위험도 평가 | 안정성 검증 |
| PEG × 회전율 | 성장 + 효율성 | 균형 평가 |
| Momentum + Value | 모멘텀 + 가치 | 수익성 극대화 |
| Equity Risk Premium | 채권 대비 주식 매력도 | 시장 환경 판단 |

# 🎯 최종 결론

복합 팩터는 **"한 가지 신호에 의존하지 않는"** 투자 철학입니다.

가치만 싼 기업은 가치 함정이 될 수 있고,
성장만 빠른 기업은 거품이 될 수 있습니다.

여러 팩터를 종합적으로 평가해서
**"정말 좋은 회사인가?"**
**"정말 싼 가격인가?"**
**"정말 성장할 회사인가?"**
를 동시에 확인할 때 최고의 투자 성과를 거둘 수 있습니다.
