# 📄 **배당 팩터 (Dividend Factor) – 5개 **

---

# # 1) **DIVIDEND_PAYOUT_RATIO (배당성향)**

### ✔ 정의

기업이 **번 돈(순이익)** 중에서 **배당금으로 얼마를 주는지** 나타내는 비율.

### ✔ 공식

```
배당성향 = (배당금 총액 / 당기순이익) × 100
```

### ✔ 해석

* **배당성향 20~40%** → 안정적
* **50%↑** → 배당 우선 정책
* **100%↑** → 번 돈보다 더 많이 배당 (지속 불가능)

### ✔ 예시

순이익 1조
배당금 3000억
→ 30% (건전)

### ✔ 주의

* 단기 실적 감소 시 의도치 않게 100% 이상으로 튈 수 있음

---

# # 2) **DIVIDEND_GROWTH_1Y (1년 배당 성장률)**

### ✔ 정의

작년 대비 **배당이 얼마나 늘었는지** 파악하는 지표.

### ✔ 공식

```
1년 배당성장률 = (현재 배당 - 작년 배당) / 작년 배당 × 100
```

### ✔ 해석

* **10%↑** 꾸준히 증가 → 배당 성장주
* **음수** → 배당 삭감 (경고)

### ✔ 예시

작년 1000원
올해 1200원
→ +20% 성장

### ✔ 주의

* 단 1년 변화만 보기 때문에 변동성이 큼

---

# # 3) **DIVIDEND_GROWTH_3Y (3년 배당 CAGR)**

### ✔ 정의

3년 동안 **평균적으로** 배당이 얼마나 성장했는지 보여주는 지표.

### ✔ 공식

```
3년 CAGR = ((현재 배당 / 3년 전 배당)^(1/3) - 1) × 100
```

### ✔ 해석

* **5~10% CAGR** → 장기 배당 성장 기업
* **일정한 상승 곡선** 확인에 적합

### ✔ 예시

3년 전 800원
현재 1200원
→ 약 +14% CAGR

### ✔ 주의

* 일회성 배당이 섞이면 CAGR 왜곡됨

---

# # 4) **DIVIDEND_CONSISTENCY (배당 지속성)**

### ✔ 정의

최근 5년 중 **몇 번 배당했는지** 보는 지표.

### ✔ 해석

* 5/5 → 매년 배당 지급 (우수)
* 3/5 → 중간배당, 불규칙
* 0~1 → 배당 정책 없음

### ✔ 예시

5년 중 5회 배당 → 꾸준한 배당 기업

### ✔ 주의

* 금액이 늘었는지 여부는 포함되지 않음 (지급 여부만)

---

# # 5) **DIVIDEND_GROWTH_YOY (전년 대비 YoY 배당성장률)**

### ✔ 정의

작년 대비 배당금이 **얼마나 증가했는지** 보는 지표 (YoY 비교).

### ✔ 공식

```
배당 YoY = (당해년도 배당 - 전년 배당) / 전년 배당 × 100
```

### ✔ 해석

* 0% → 배당 유지
* +10% → 성장
* -10% → 배당 삭감 위험

### ✔ 예시

전년 1500원
올해 1800원
→ +20% YoY 성장

### ✔ 주의

* 최근 실적의 영향을 강하게 받음
* YoY만 보고 급격한 성장이라고 착각하면 위험

# 🎯 **최종 요약**

| 팩터                        | 의미                 | 핵심 포인트      |
| ------------------------- | ------------------ | ----------- |
| **DIVIDEND_PAYOUT_RATIO** | 이익 중 배당 비율         | 지속 가능성 판단   |

| **DIVIDEND_GROWTH_3Y**    | 3년 평균 배당 성장률(CAGR) | 장기 성장성      |

| **DIVIDEND_GROWTH_YOY**   | 전년 대비 성장           | 단기 성과 확인    |

# 🎯 **최종 결론**

배당 팩터들은 크게 **두 가지**를 평가한다:

1. 📈 **배당이 꾸준히 늘고 있는가? (성장성)**
2. 🔒 **배당을 오랫동안 꾸준히 주고 있는가? (지속성/안정성)**

특히:

* **배당성향(Payout Ratio)**
* **3년 CAGR**
* **배당 지속성(5년 지급 기록)**

이 세 가지를 함께 보면 **"배당주로서의 품질"**을 정확히 판단할 수 있다.
