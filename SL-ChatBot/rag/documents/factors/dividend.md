# 배당 팩터 (Dividend Factor)

## 개념

**배당 팩터**는 **배당금을 많이 지급하는 기업에 투자하는 팩터**입니다. 정기적인 현금 수익을 추구합니다.

> 💡 **핵심 아이디어**: 정기 배당금 + 완만한 주가 상승 = 안정적 수익

---

## 핵심 지표

### 1. 배당수익률

**의미**: 투자금에 대한 배당금의 비율

```
배당수익률 = 연간 배당금 / 주가 × 100%
```

**해석**:
- 배당수익률 > 3%: 충분한 수익
- 배당수익률 2~3%: 평균 수준
- 배당수익률 < 2%: 저수익

**예시**:
```
주가: 50,000원
연간 배당금: 1,500원

배당수익률 = 1,500 / 50,000 × 100 = 3%
→ 매년 3% 배당금 수익
```

### 2. 배당성향

**의미**: 순이익 중 배당금으로 지급하는 비율

```
배당성향 = 배당금 / 순이익 × 100%
```

**해석**:
- 배당성향 30~50%: 적정 (지속 가능)
- 배당성향 > 60%: 높음 (감소 위험)
- 배당성향 < 20%: 낮음 (증가 여지)

**예시**:
```
순이익: 100억원
배당금: 30억원

배당성향 = 30 / 100 × 100 = 30%
→ 적정 수준 (지속 가능성 높음)
```

### 3. 배당 성장률

**의미**: 배당금이 해마다 증가하는 정도

```
배당 성장률 = (현재 배당금 - 전년 배당금) / 전년 배당금 × 100%
```

**해석**:
- 5년 지속 증가: 매우 우수
- 연 5% 이상 증가: 우수
- 불규칙하거나 감소: 주의

**예시**:
```
2020년: 1,000원
2021년: 1,050원 (5% 증가)
2022년: 1,103원 (5% 증가)
2023년: 1,158원 (5% 증가)
2024년: 1,216원 (5% 증가)

→ 배당 성장 기록 우수 ✅
```

---

## 장점 ✅

| 장점 | 설명 |
|------|------|
| **정기 수익** | 매년 배당금 수령 |
| **안정성** | 배당 기업은 실적 안정적 |
| **심리 안정** | 주가 변동 신경 덜 쓸 수 있음 |
| **복리 효과** | 배당금 재투자로 복리 |
| **인플레이션 헤지** | 배당금 증가로 인플레이션 대응 |

---

## 단점 ❌

| 단점 | 설명 |
|------|------|
| **주가 상승 제약** | 이익을 배당으로 나누므로 성장 제약 |
| **금리 민감** | 금리 인상 시 배당 매력 감소 |
| **배당 감소 위험** | 실적 부진 시 배당 삭감 |
| **배당세** | 배당금에 세금 부과 (15.4%) |
| **인플레이션** | 배당금 증가가 인플레이션 못 따라갈 수 있음 |

## 언제 사용할까?

### ✅ 좋은 경우

- 은퇴자 (정기 현금 필요)
- 보수적 투자자
- 장기 보유 (배당 재투자)
- 낮은 금리 시기
- 안정적 수익 추구

### ❌ 피해야 할 경우

- 높은 금리 시기
- 고성장 추구
- 단기 수익 필요
- 배당 감소 신호 기업
- 세금 부담 큰 경우

## 실전 분석 예시

### 포스코 (배당주)

```
배당수익률: 3.5%
배당성향: 40%
배당 성장: 연 5% 이상 (5년 지속)

판단: 우수한 배당주 ✅
- 충분한 배당 수익
- 적정 배당성향 (지속 가능)
- 배당 증가 추세
```

### 현대제철 (배당주)

```
배당수익률: 4%
배당성향: 35%
배당 성장: 5년 연속 증가

판단: 매력적 배당주 ✅
```

## 배당주 체크리스트

배당주 선택 전 확인하세요:

- [ ] 배당수익률 > 2.5%?
- [ ] 배당성향 30~50%?
- [ ] 배당 성장 기록 5년?
- [ ] ROE > 10%? (배당 여력)
- [ ] 부채비율 < 100%? (배당 지속성)
- [ ] 산업 경기 안정?
- [ ] 배당 감소 신호 없음?
- [ ] 3년 이상 보유 가능?

## 배당 기업 vs 성장 기업

| 항목 | 배당주 | 성장주 |
|------|--------|--------|
| **배당** | 높음 | 낮음 |
| **주가 상승** | 완만 | 큼 |
| **변동성** | 낮음 | 높음 |
| **적합자** | 은퇴자 | 청년 투자자 |
| **위험도** | 낮음 | 높음 |

## FAQ

**Q: 배당주만 사도 되나요?**
A: 아닙니다. 다양한 자산 배분 권장 (배당주 + 성장주)

**Q: 배당금은 세금을 내야 하나요?**
A: 네, 15.4% 배당세 부과됩니다.

**Q: 배당을 받지 않는 게 낫나요?**
A: 아닙니다. 수익 창출 목표면 배당금 받기 권장

**Q: 배당이 줄어들면 팔아야 하나요?**
A: 배당 감소가 일시적인지 구조적인지 분석 후 결정

**Q: 금리가 올라가면?**
A: 배당수익률 매력 감소 → 배당주 회피 가능
