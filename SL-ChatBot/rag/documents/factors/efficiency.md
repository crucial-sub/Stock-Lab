
# ⚙️ **EFFICIENCY 팩터 (3개) **

효율성(Efficiency) 팩터는
기업이 **자산·운전자본·매출채권 등을 얼마나 효율적으로 사용해 매출을 만들어내는지** 평가하는 지표다.

> 💡 핵심: “기업이 가진 자원을 얼마나 잘 굴리고 있는가?”

---

# 🔵 **1) RECEIVABLES_TURNOVER (매출채권회전율)**

### ✔ 정의

기업이 **외상매출(매출채권)** 을 얼마나 빨리 회수하는지 보여주는 지표.
→ 현금회수 속도 평가.

### ✔ 공식

```
매출채권회전율 = 매출액 / 평균 매출채권
```

### ✔ 해석

* 높을수록 → 외상매출 회수 빠름 → 현금흐름 건강
* 낮을수록 → 회수 지연 → 부실채권 위험

### ✔ 예시

매출 1조
평균 매출채권 2,000억
→ 회전율 = 5 (연 5회 회수)

### ✔ 주의

* B2B 기업은 회전율이 낮아도 정상
* IT·플랫폼 기업은 회전율 높음

---

# 🔵 **2) WORKING_CAPITAL_TURNOVER (운전자본회전율)**

### ✔ 정의

재고·매출채권·매입채무 등 **운전자본의 효율성**을 평가.
→ “투입한 운전자본 대비 얼마나 매출이 나오는가?”

### ✔ 공식

```
운전자본회전율 = 매출액 / 운전자본
```

※ 운전자본 = 유동자산 – 유동부채

### ✔ 해석

* 높을수록 → 운전자본 효율적
* 낮으면 → 재고·채권 등 묶인 자금 많음 → 비효율

### ✔ 예시

매출 2조
운전자본 5,000억
→ 회전율 = 4 (양호)

### ✔ 주의

* 음수 운전자본 구조(쿠팡, 마트 업종)는 회전율 해석이 어려움

---

# 🔵 **3) FIXED_ASSET_TURNOVER (고정자산회전율)**

### ✔ 정의

공장·설비·기계 같은 **고정자산이 매출로 얼마나 잘 연결되는지** 평가.

### ✔ 공식

```
고정자산회전율 = 매출액 / 평균 고정자산
```

### ✔ 해석

* 높음 → 설비 투자 효율 좋음
* 낮음 → 설비 투자 대비 매출이 적음(비효율)

### ✔ 예시

고정자산 1조 → 매출 3조
→ 회전율 = 3 (효율적)

### ✔ 주의

* 제조업과 IT 기업은 기준이 크게 다름
* 신공장·증설 직후에는 회전율이 낮아질 수 있음

---

# 🎯 **EFFICIENCY 팩터 요약**

| 팩터      | 핵심 의미      | 무엇을 평가? |
| ------- | ---------- | ------- |
| 매출채권회전율 | 외상매출 회수 속도 | 현금 회수능력 |
| 운전자본회전율 | 운전자본 효율성   | 자금 운영력  |
| 고정자산회전율 | 설비 효율      | 생산활동 효율 |

---

# 🎯 **최종 결론**

EFFICIENCY 팩터는
기업이 가진 자원을 **얼마나 효율적으로 활용하고 있는지** 보여주는 핵심 지표다.

특히 다음 두 가지가 실전에서 중요하다:

* **매출채권회전율** → 현금 회수 속도
* **고정자산회전율** → 설비 투자 효율성

효율성이 높은 기업은
적은 자원으로도 높은 매출을 내고,
위기 상황에서도 현금흐름이 안정적이다.
