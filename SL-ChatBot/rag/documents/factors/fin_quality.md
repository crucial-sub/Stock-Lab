# 🏛 **FINANCIAL QUALITY 팩터 (6개) **

재무 품질(Financial Quality) 팩터는 기업의
**수익 품질·레버리지 건전성·현금 흐름의 질·위험 가능성** 등을 평가한다.

> 💡 핵심 질문:
> “이 회사의 재무 구조는 얼마나 건강한가?”

---

# 🔵 **1) DUPONT_ROE (듀폰 분석 ROE)**

### ✔ 정의

ROE(자기자본수익률)를 **3가지 요인으로 분해**하여
기업의 수익성이 어떤 요소에 의해 만들어지는지 분석하는 기법.

### ✔ 공식

```
DuPont ROE = 순이익률 × 자산회전율 × 재무레버리지
```

### ✔ 해석

* 순이익률이 높다 → 본업 수익성 좋음
* 자산회전율 높다 → 효율적인 사업 구조
* 레버리지 높다 → 부채 활용으로 ROE 상승

### ✔ 예시

ROE 12%가

* 순이익률 4%
* 회전율 1.0
* 레버리지 3

→ “수익성 낮지만 레버리지로 ROE가 높아 보임”

### ✔ 주의

ROE가 높아도 **레버리지로만 올라가는 경우 위험**.

---

# 🔵 **2) OPERATING_LEVERAGE (영업 레버리지)**

### ✔ 정의

매출 증가가 영업이익 증가에 얼마나 민감한지.
→ 고정비 구조가 많은 기업에서 중요.

### ✔ 공식

```
영업 레버리지 = 영업이익 변화율 / 매출 변화율
```

### ✔ 해석

* 2.0↑ → 매출 조금만 증가해도 이익이 크게 증가
* 경기가 좋아지면 폭발적 성장 가능
* 경기가 나빠지면 급격히 악화됨(양날의 검)

### ✔ 예시

매출 +10% → 영업이익 +25%
= 영업 레버리지 2.5 (높음)

### ✔ 주의

고정비 비중이 높은 산업(항공, 제조)은 변동성이 매우 큼.

---

# 🔵 **3) FINANCIAL_LEVERAGE (재무 레버리지)**

### ✔ 정의

기업이 부채를 얼마나 활용하는지.
ROE를 높여주는 효과도 있지만 위험도 증가.

### ✔ 공식

```
재무 레버리지 = 총자산 / 자기자본
```

### ✔ 해석

* 3.0↑ → 부채 의존도 높음
* 1.5~2.5 → 일반적인 안정 범위
* 지나치면 이자비용 부담 폭발

### ✔ 예시

총자산 10조 / 자기자본 3조 = 3.33
→ 재무 레버리지 높음

### ✔ 주의

금리 상승기에는 부채 많은 기업이 크게 흔들림.

---

# 🔵 **4) CASH_CONVERSION_EFFICIENCY (현금전환효율성)**

### ✔ 정의

순이익이 **얼마나 효율적으로 현금흐름으로 전환되는지** 보는 지표.

### ✔ 공식

```
CCE = 영업현금흐름 / 순이익
```

### ✔ 해석

* 1.0↑ → 이익이 충분히 현금으로 들어옴 (건전한 회사)
* 1.0↓ → 이익이 회계상 숫자일 가능성(주의)
* 음수 → 위험 신호

### ✔ 예시

순이익 5,000억, OCF 6,000억 → CCE = 1.2
→ 현금 창출 능력 매우 우수

### ✔ 주의

* 성장기업은 재고 확대 때문에 일시적으로 CCE가 낮을 수 있음
* 매출채권 증가가 CCE 하락 원인일 수 있음

---

# 🔵 **5) ZMIJEWSKI_SCORE (즈미예프스키 파산 위험 점수)**

### ✔ 정의

기업의 **파산 위험**을 예측하는 회계 기반 모델.

### ✔ 공식

(대표 형태)

```
Z = -4.3 
    - 4.5 × 부채비율 
    + 5.7 × 순이익률 
    - 0.004 × 유동비율
```

### ✔ 해석

* Z > 0 → 파산 위험 높음
* Z < 0 → 안정적

### ✔ 예시

Z = 1.2 → 재무 위험 높음
Z = -2.5 → 매우 안정적

### ✔ 주의

* 제조업 외 산업은 해석 기준 달라질 수 있음
* 단독 사용보다는 참고용으로 활용

---

# 🔵 **6) ASSET_QUALITY (자산 품질)**

### ✔ 정의

대손충당금, 부실채권, 감가상각 등
자산이 **얼마나 건전하게 관리되고 있는지** 평가하는 지표.

### ✔ 주요 항목 포함

* 부실채권비율
* 고정자산비중
* 대손충당금 적립 여부
* 자산 손상처리 이슈

### ✔ 해석

* 자산 품질 높음 → 안정적 기업
* 자산 품질 낮음 → 부실 위험 증가

### ✔ 예시

부실채권비율 급증 → 자산 품질 악화 신호

### ✔ 주의

금융업(은행/보험)의 자산 품질은 일반 기업과 해석 기준 다름.

# 🎯 **FIN_QUALITY 팩터 요약표**

| 팩터                         | 의미           | 핵심 포인트       |
| -------------------------- | ------------ | ------------ |
| DuPont ROE                 | ROE 구성요소 분석  | 수익성·효율성·레버리지 |
| Operating Leverage         | 매출 대비 이익 민감도 | 고정비 구조       |
| Financial Leverage         | 부채 활용도       | 위험 + ROE 영향  |
| Cash Conversion Efficiency | 이익 → 현금 전환력  | 현금 기반 재무 건전성 |
| Zmijewski Score            | 파산 위험        | 위험 조기 탐지     |
| Asset Quality              | 자산 건전성       | 부실 가능성       |

---

# 🎯 **최종 결론**

FIN_QUALITY 팩터는 기업의 **재무 체질**을 평가하기 때문에
가치·성장·수익성 등 다른 팩터보다 **리스크 관리에 매우 중요**하다.

특히 중요 핵심 3개:

* **Cash Conversion Efficiency** → “이익이 진짜 돈인가?”
* **Financial Leverage** → 부채 리스크
* **DuPont ROE** → ROE의 진짜 원인 분석

FIN_QUALITY는
👉 “실적은 좋아보이는데 위험한 기업”
👉 “조용히 위험이 쌓이고 있는 기업”
을 걸러내는 데 가장 강력한 팩터다.
