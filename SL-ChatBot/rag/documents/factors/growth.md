

# 📈 **성장 팩터(Growth Factor) – 15개 완전 정리**

성장 팩터는 기업의 **매출·이익·현금흐름·자산이 얼마나 빨리 커지고 있는지** 확인하는 지표다.

“성장하는 기업인지?”를 가장 직관적으로 판단하는 카테고리.

---

# # 1) **REVENUE_GROWTH_1Y (1년 매출 성장률)**

### ✔ 정의

작년 대비 매출이 얼마나 늘었는지.

### ✔ 공식

```
(당기 매출 - 전기 매출) / 전기 매출 × 100
```

### ✔ 해석

* +10% 이상 → 성장 기업
* 0% 근처 → 정체
* 음수 → 매출 감소 (경고)

### ✔ 예시

전기 10조 → 당기 11조
= +10%

---

# # 2) **REVENUE_GROWTH_3Y (3년 매출 CAGR)**

### ✔ 정의

3년간 매출이 평균적으로 얼마나 성장했는지.

### ✔ 공식

```
((당기 매출 / 3년 전 매출)^(1/3) - 1) × 100
```

### ✔ 해석

* +8~15% → 중장기 성장 우수
* 안정적 장기 성장을 평가할 때 핵심

---

# # 3) **EARNINGS_GROWTH_1Y (1년 순이익 성장률)**

### ✔ 정의

전년 대비 순이익 증가율.

### ✔ 해석

* +20% 이상 → 강한 실적 모멘텀
* -10%↓ → 실적 악화

### ✔ 예시

순이익 2조 → 2.4조
+20%

---

# # 4) **EARNINGS_GROWTH_3Y (3년 순이익 CAGR)**

### ✔ 정의

3년간 순이익의 평균 성장률.

### ✔ 해석

* 장기 실적 성장 여부 판단
* 단기 변동성을 제거

---

# # 5) **OCF_GROWTH_1Y (영업현금흐름 성장률)**

### ✔ 정의

영업활동에서 들어오는 현금(OCF)이 1년간 얼마나 늘었는지.

### ✔ 의미

* 매출 성장보다 더 신뢰도 높은 지표
* 현금창출력이 강화되는 기업은 강함

### ✔ 예시

5,000억 → 6,000억
= +20%

---

# # 6) **ASSET_GROWTH_1Y (자산 성장률)**

### ✔ 정의

총자산이 얼마나 증가했는지.

### ✔ 해석

* +10%↑ → 적극적 투자/성장
* 0% 근처 → 정체
* 음수 → 구조조정 가능성

---

# # 7) **BOOK_VALUE_GROWTH_1Y (순자산 성장률)**

### ✔ 정의

기업 순자산(BV)이 얼마나 증가했는지.

### ✔ 예시

순자산 5조 → 5.5조
= +10%

### ✔ 의미

기업가치의 장기적 증가 여부 판단 가능

---

# # 8) **SUSTAINABLE_GROWTH_RATE (지속가능 성장률)**

### ✔ 정의

기업이 **외부 조달 없이** 내부 자본만으로 성장 가능한 최대 성장률.

### ✔ 공식

```
SGR = ROE × (1 - 배당성향)
```

### ✔ 해석

* ROE가 높고
* 배당성향이 낮으면
  → **높은 내부 성장 가능** 기업

---

# # 9) **REINVESTMENT_RATE (재투자율)**

### ✔ 정의

이익 중에서 성장(설비 투자 등)에 얼마나 재투자하는지.

### ✔ 공식

```
(CAPEX - 감가상각) / 순이익
```

### ✔ 해석

* 높은 재투자율 → 미래 성장을 위한 투자 적극적
* 낮은 재투자율 → 성숙기 기업일 가능성

---

# # 10) **REVENUE_GROWTH_YOY (매출 YoY 성장률)**

### ✔ 정의

전년 동기와 비교한 매출 증가율.

### ✔ 해석

* 계절성 영향을 배제할 때 유용

---

# # 11) **REVENUE_GROWTH_QOQ (매출 분기대비 성장률)**

### ✔ 정의

직전 분기 대비 매출 증가율.

### ✔ 해석

* 단기 모멘텀 파악
* 정상적 분기 변동성 고려 필요

---

# # 12) **OPERATING_INCOME_GROWTH_YOY (영업이익 YoY 증가율)**

### ✔ 정의

전년 대비 영업이익 증가율.

### ✔ 해석

* 본업의 수익성 개선 여부 판단
* 매출보다 **더 신뢰도가 높은 성장 지표**

---

# # 13) **NET_INCOME_GROWTH_YOY (순이익 YoY 증가율)**

### ✔ 정의

전년 대비 순이익 증가율.

### ✔ 해석

* 비경상적 이익/손실의 영향을 받을 수 있음

---

# # 14) **EPS_GROWTH_YOY (주당순이익 증가율)**

### ✔ 정의

EPS(주당순이익)의 전년 대비 증가율.

### ✔ 의미

* EPS가 늘어난다는 건
  → "실제 한 주당 가치를 기업이 키우고 있다"는 것

---

# # 15) **ASSET_GROWTH_YOY (총자산 YoY 증가율)**

### ✔ 정의

1년 동안 총자산이 얼마나 늘었는지.

### ✔ 의미

* 확장·설비투자·M&A 반영
* 성장 속도 판단 가능

---

# 🎯 **성장 팩터 최종 요약**

| 팩터                        | 핵심 의미           |
| ------------------------- | --------------- |
| **매출 성장(REVENUE)**        | 기업의 외형 성장       |
| **순이익 성장(EARNINGS, EPS)** | 실질적인 수익 성장      |
| **현금흐름 성장(OCF)**          | 실제 돈이 늘어나는지     |
| **자산·순자산 성장(ASSET/BV)**   | 기업 규모 확대        |
| **지속가능 성장(SGR)**          | 내부적으로 감당 가능한 성장 |
| **재투자율**                  | 장기 성장 의지        |
| **YoY, QoQ**              | 분기/연간 성장 속도     |

---

# 🎯 **최종 결론**

성장 팩터는 기업의 **미래가치가 커지고 있는지**를 확인하는 데 핵심이다.

특히 다음 5개가 가장 중요하다:

* **매출 성장률(REVENUE_GROWTH)**
* **순이익 성장률(EARNINGS_GROWTH)**
* **EPS 성장률(EPS_GROWTH)**
* **OCF 성장률(현금 기반 성장)**
* **지속가능 성장률(SGR)**

이 5가지를 함께 보면
👉 “이 기업이 앞으로 더 커질 기업인가?”
직관적으로 판단할 수 있다.
