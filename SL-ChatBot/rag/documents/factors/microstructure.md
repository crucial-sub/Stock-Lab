# 🧩 **MICROSTRUCTURE 팩터 (5개) — 초보자용 완전 정리**

Microstructure(시장 미시구조) 팩터는
**거래량·체결 강도·유동성·일중 움직임의 질**을 측정하는 지표다.

> 💡 핵심 포인트: “이 종목이 실제로 잘 거래되고 있는가?”
> → 급등·급락, 비유동성, 세력 매집 여부 등을 파악할 수 있음.

---

# 🔵 **1) AMIHUD_ILLIQUIDITY (아미후드 비유동성)**

### ✔ 정의

거래량 대비 가격 변동폭을 이용해 **유동성 부족 정도**를 측정하는 지표.

### ✔ 공식

```
아미후드 비유동성 = |일간 수익률| / 거래대금
```

### ✔ 해석

* 값이 **높다** → 유동성 매우 낮음 (조금만 거래해도 가격 출렁)
* 값이 **낮다** → 거래가 원활함, 체결 품질 높음

### ✔ 예시

* 1% 변동에 거래대금 1억 → 0.01
* 1% 변동에 거래대금 1천만 → 0.1 (10배 나쁨)

### ✔ 주의

* 중소형주에서 특히 중요
* 거래대금이 지극히 낮은 종목은 왜곡됨

---

# 🔵 **2) INTRADAY_VOLATILITY (일중 변동성)**

### ✔ 정의

같은 날 **고가–저가 차이**로 일중 가격 출렁임을 측정.

### ✔ 공식

```
일중 변동성 = (고가 - 저가) / 종가
```

### ✔ 해석

* 높을수록 당일 변동성이 큼
  → 테마주·세력주에서 자주 높게 나타남
* 낮을수록 안정적

### ✔ 예시

* 고가 10,000 / 저가 9,200 / 종가 9,800
  → (10,000 - 9,200)/9,800 = 약 8.1%

### ✔ 주의

* 장중 이슈(뉴스, 공시)에 매우 민감
* 거래량 부족한 종목은 왜곡됨

---

# 🔵 **3) VOLUME_PRICE_TREND (거래량-가격 추세, VPT)**

### ✔ 정의

가격 변화와 거래량을 결합해 **추세의 힘**을 측정.

### ✔ 공식

```
VPT = 이전 VPT + (금일 수익률 × 금일 거래량)
```

### ✔ 해석

* VPT↑ → 상승 추세에 거래량 동반(건강한 상승)
* VPT↓ → 하락 추세 강화

### ✔ 예시

주가 +2%, 거래량 500만
→ 매수세 강하게 유입된 신호

### ✔ 주의

* 단기 급등주에서는 거래량 폭증으로 왜곡 가능
* 장기 추세 확인에 유리

---

# 🔵 **4) EASE_OF_MOVEMENT (가격 이동 용이성, EMV)**

### ✔ 정의

적은 거래량으로도 가격이 얼마나 쉽게 움직이는지 측정한 지표.

### ✔ 공식

```
EMV = (중심가격 변화 / 거래량) × 스케일 보정
```

※ 중심가격 = (고가 + 저가) / 2

### ✔ 해석

* EMV↑ → 적은 거래량에도 가격이 쉽게 올라감
  → “매수세가 가벼운 힘만으로도 가격을 밀어 올림”
* EMV↓ → 거래량 대비 가격이 잘 안 움직임

### ✔ 예시

거래량 적은데도 주가 급등 → EMV 매우 높음

### ✔ 주의

* 장중 거래량 왜곡 시 민감하게 움직임
* 단독 사용보다는 보조지표로 활용

---

# 🔵 **5) FORCE_INDEX (가격 압력 지수)**

### ✔ 정의

가격 변화 + 거래량 두 요소로
**매수·매도 압력이 얼마나 강한지** 측정하는 지표.

### ✔ 공식

```
Force Index = (금일 종가 - 전일 종가) × 거래량
```

### ✔ 해석

* 양수 & 큰 값 → 매수 압력 강함
* 음수 & 큰 절대값 → 매도 압력 강함

### ✔ 예시

종가 +3% 상승, 거래량 폭증
→ 강한 매수세 진입

### ✔ 주의

* 단기 급등주는 Force Index 과도하게 튈 수 있음
* 보조지표로 추세 확인에 좋음

# 🎯 **MICROSTRUCTURE 팩터 요약**

| 팩터                      | 핵심 의미     | 무엇을 파악?         |
| ----------------------- | --------- | --------------- |
| **AMIHUD ILLIQUIDITY**  | 유동성 부족 정도 | 가격이 쉽게 출렁이는지    |
| **INTRADAY VOLATILITY** | 일중 변동성    | 장중 위험 수준        |
| **VPT**                 | 거래량+가격 추세 | 상승/하락의 강도       |
| **EMV**                 | 가격 이동 용이성 | 적은 거래로 가격 올라가는가 |
| **Force Index**         | 매수/매도 압력  | 세력/모멘텀 흐름       |

# 🎯 **최종 결론**

Microstructure 팩터는
기업의 “가치”가 아니라 **주가의 움직임 품질**을 보여주는 지표다.

특히 다음에서 중요함:

✔ 단기 트레이딩
✔ 테마 급등 종목
✔ 매집 여부 파악
✔ 비유동성 종목 위험 관리

> 종목을 단순히 “오르는 이유”가 아니라
> “얼마나 건강하게 오르고 있는지” 판단할 때 필수다.
