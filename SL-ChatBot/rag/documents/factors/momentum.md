# 모멘텀 팩터 (Momentum Factors)

## 개요
모멘텀 투자는 최근 수익률이 높은 주식이 단기적으로 계속 상승하는 경향을 활용하는 전략입니다.

## 주요 팩터

### 1. 단기 모멘텀 (1-3개월)
- **1개월 수익률**: 초단기 추세
- **3개월 수익률**: 단기 추세
- **기준값**: 10% 이상
- **용도**: 단타, 스윙 트레이딩

### 2. 중기 모멘텀 (6개월)
- **6개월 수익률**: 중기 추세
- **기준값**: 15-20% 이상
- **용도**: 추세 추종 전략의 핵심

### 3. 장기 모멘텀 (12개월)
- **12개월 수익률**: 장기 추세
- **기준값**: 20-30% 이상
- **주의**: 최근 1개월은 제외 (역모멘텀)

### 4. 상대강도 (Relative Strength)
- **정의**: 시장 대비 상대 수익률
- **공식**: `개별주 수익률 - 시장지수 수익률`
- **기준**: 양수 (시장 대비 초과 수익)

### 5. 거래량 모멘텀
- **평균 거래량 대비 비율**
- **급증 기준**: 평균 대비 1.5-2배 이상
- **의미**: 매수세 유입

## 전략 조합 예시

### 기본 모멘텀 전략
```json
{
  "buy_conditions": [
    {"factor": "3개월수익률", "operator": ">", "value": 10},
    {"factor": "6개월수익률", "operator": ">", "value": 15}
  ]
}
```

### 중장기 모멘텀 전략
```json
{
  "buy_conditions": [
    {"factor": "6개월수익률", "operator": ">", "value": 20},
    {"factor": "12개월수익률", "operator": ">", "value": 30}
  ]
}
```

### 모멘텀 + 퀄리티 전략
```json
{
  "buy_conditions": [
    {"factor": "6개월수익률", "operator": ">", "value": 15},
    {"factor": "ROE", "operator": ">", "value": 12},
    {"factor": "부채비율", "operator": "<", "value": 100}
  ]
}
```

### 듀얼 모멘텀 전략
```json
{
  "buy_conditions": [
    {"factor": "6개월수익률", "operator": ">", "value": 0},
    {"factor": "12개월수익률", "operator": ">", "value": 0}
  ]
}
```
- 절대 모멘텀 (양수 수익률) + 상대 모멘텀 조합

## 모멘텀 측정 기간별 특징

### 1개월 (초단기)
- **장점**: 빠른 반응
- **단점**: 노이즈 많음, 거래비용 높음
- **적합**: 변동성 큰 시장

### 3개월 (단기)
- **장점**: 단기 추세 포착
- **단점**: 여전히 변동성
- **적합**: 스윙 트레이딩

### 6개월 (중기) ⭐ 가장 효과적
- **장점**: 추세와 노이즈 균형
- **단점**: 반응 속도
- **적합**: 추세 추종 전략

### 12개월 (장기)
- **장점**: 장기 추세 확인
- **단점**: 반전 신호 늦음
- **적합**: 저빈도 리밸런싱

## 모멘텀 크래시 방지

### 1. 손절매 설정
```
- 진입가 대비 -10% 손절
- 최고점 대비 -15% 손절
```

### 2. 변동성 필터
```json
{
  "buy_conditions": [
    {"factor": "6개월수익률", "operator": ">", "value": 15},
    {"factor": "변동성", "operator": "<", "value": 40}
  ]
}
```

### 3. 거래량 확인
```json
{
  "buy_conditions": [
    {"factor": "6개월수익률", "operator": ">", "value": 15},
    {"factor": "평균거래대금", "operator": ">", "value": 1000000000}
  ]
}
```

### 4. 퀄리티 필터
```json
{
  "buy_conditions": [
    {"factor": "6개월수익률", "operator": ">", "value": 15},
    {"factor": "ROE", "operator": ">", "value": 10},
    {"factor": "부채비율", "operator": "<", "value": 150}
  ]
}
```

## 시장 상황별 전략

### 상승장 (Bull Market)
- **전략**: 공격적 모멘텀
- **기간**: 3-6개월
- **기준**: 높은 수익률 (20% 이상)

### 하락장 (Bear Market)
- **전략**: 방어적 접근
- **기간**: 절대 모멘텀 (양수만)
- **기준**: 현금 비중 확대

### 횡보장 (Sideways)
- **전략**: 짧은 기간 모멘텀
- **기간**: 1-3개월
- **기준**: 섹터 로테이션

## 섹터 로테이션 활용

### 경기 순환 단계별
```
확장 초기: 금융, IT
확장 중기: 산업재, 소재
확장 후기: 에너지, 원자재
수축 초기: 필수소비재, 헬스케어
수축 중기: 유틸리티, 방어주
```

## 주의사항

1. **과거 수익 ≠ 미래 수익**
   - 모멘텀은 지속되지만 반전 위험
   - 과도한 상승 후 조정

2. **거래비용**
   - 빈번한 매매로 비용 증가
   - 리밸런싱 주기 최적화 필요

3. **크라우딩 리스크**
   - 많은 투자자가 동일 전략 사용
   - 급격한 반전 가능성

4. **버블 위험**
   - 과열 국면에서 높은 수익률
   - 갑작스런 붕괴 위험

5. **시장 환경**
   - 추세장에서 효과적
   - 횡보장에서 손실 가능

## 성과 측정

### 승률 (Win Rate)
- 목표: 60% 이상

### 손익비 (Profit/Loss Ratio)
- 목표: 2:1 이상

### MDD (Maximum Drawdown)
- 목표: -30% 이내

### 샤프 비율
- 목표: 1.0 이상

## 고급 모멘텀 전략

### 1. 잔여 모멘텀 (Residual Momentum)
- 시장 베타 제거 후 초과 수익
- 순수 종목 고유 모멘텀

### 2. 52주 신고가 전략
```json
{
  "buy_conditions": [
    {"factor": "현재가", "operator": ">=", "value": "52주최고가_95%"}
  ]
}
```

### 3. 이동평균 돌파 전략
```
골든크로스: 단기 MA > 장기 MA
데드크로스: 단기 MA < 장기 MA
```

## 백테스트 체크리스트

- [ ] 다양한 시장 환경 테스트
- [ ] 거래비용 반영 (0.3% 왕복)
- [ ] 슬리피지 고려 (0.1%)
- [ ] 리밸런싱 주기 최적화
- [ ] 손절매 규칙 설정

- [ ] 포지션 사이징
