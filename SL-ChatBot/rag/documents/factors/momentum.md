# ✨ 주요 모멘텀 지표 설명
## 1. MOMENTUM (1M / 3M / 6M / 12M)
✔ 정의

기간별 주가 상승률을 측정한 지표.

✔ 공식
(현재 가격 - N개월 전 가격) / N개월 전 가격 × 100

✔ 해석

양수(+) → 상승 추세

음수(-) → 하락 추세

6M / 12M 모멘텀이 가장 보편적으로 사용됨

✔ 예시

6M 모멘텀 +25% → 강한 상승 추세

## 2. RETURN (1M / 3M / 6M / 12M)
✔ 정의

모멘텀과 유사하지만, 더 넓은 의미의 총수익률(return) 을 의미하기도 함.

✔ 공식
(현재 가격 - 과거 가격) / 과거 가격 × 100

✔ 해석

Return = 단순 상승률

Momentum = Return을 기반으로 만든 팩터

✔ 예시

12M Return +40% → 1년 동안 강한 상승

## 3. DISTANCE_FROM_52W_HIGH
✔ 정의

현재 가격이 52주 최고가에서 얼마나 떨어져 있는지.

✔ 공식
(52주 고가 - 현재가) / 52주 고가

✔ 해석

값이 0에 가까울수록 고점 근처 → 상승 추세

값이 높을수록 고점 대비 멀어짐 → 지친 흐름

✔ 예시

0.08 → 고점 근처

## 4. DISTANCE_FROM_52W_LOW
✔ 정의

현재 가격이 52주 최저가에서 얼마나 올라왔는지.

✔ 해석

값이 높을수록 바닥 대비 꽤 상승 → 모멘텀 살아남

값이 낮을수록 아직 바닥권

## 5. DAYS_FROM_52W_HIGH
✔ 정의

52주 최고가 이후 몇 일이 지났는지 측정.

✔ 해석

0일 → 방금 최고가 갱신

적을수록 → 강한 모멘텀 유지

많을수록 → 추세가 오래됨

## 6. DAYS_FROM_52W_LOW
✔ 정의

52주 최저가 이후 몇 일이 지났는지 측정.

✔ 해석

낮을수록 바닥권

상승 초기 흐름 포착에 좋음

## 7. RELATIVE_STRENGTH (상대강도)
✔ 정의

해당 종목이 시장/업종 대비 얼마나 강한지.

✔ 해석

1보다 크면 → 시장보다 강함

1보다 작으면 → 시장보다 약함

모멘텀 전략의 핵심 지표

✔ 예시

RS > 1.2 → 강한 상대강세

## 8. VOLUME_MOMENTUM (거래량 모멘텀)
✔ 정의

가격 상승이 거래량 증가와 함께 일어나는지 측정.

✔ 해석

거래량 증가 + 주가 상승 = 진짜 모멘텀

거래량 감소 + 상승 = 약한 모멘텀

# 📊 모멘텀 전략의 강점
강점	설명
효율 검증됨	학술 연구·백테스트 모두 높은 성과
추세 기반	시장의 심리적 흐름을 그대로 탑승
성장주와 궁합 최고	상승하는 성장주 포착 용이
기술적 지표와 연계 용이	MACD, RSI 등과 조합 강력
# ⚠️ 모멘텀 전략의 단점
단점	설명
고점 추격 리스크	과열 구간 진입 시 큰 조정 위험
변동성 증가	급등 후 급락 가능성 존재
이벤트 민감	뉴스·실적 발표에 취약
# 🤝 함께 보면 좋은 지표
조합	이유
모멘텀 + 거래량(VOLUME)	추세의 신뢰도 강화
모멘텀 + 퀄리티(QUALITY)	허약한 잡주 배제
모멘텀 + 변동성(VOLATILITY)	리스크 조절 용이
# ✔ 체크리스트

 3M / 6M / 12M 모멘텀 양수인가?

 시장 대비 상대강도(RS)가 강한가?

 거래량이 증가하고 있는가?

 고점 대비 거리(52W High)가 너무 크지 않은가?

 최근 급등/급락 이후인가?

# 🎯 최종 결론

모멘텀 팩터는
**“상승 추세를 빠르게 잡고, 그 흐름을 최대한 길게 가져가는 전략”**입니다.

단순한 기술적 지표를 넘어
기간 상승률 + 상대강도 + 거래량 + 52주 위치를 함께 보면
모멘텀의 질과 신뢰도를 훨씬 더 정확하게 판단할 수 있습니다.
