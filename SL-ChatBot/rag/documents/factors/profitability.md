# 💰 **PROFITABILITY 팩터 **

수익성 팩터는 기업이 **얼마나 효율적으로 돈을 버는지** 측정하는 지표들이다.

> 💡 핵심 포인트: “이익을 얼마나 잘 남기는 회사인가?”
> → 장기적으로 우량 기업일수록 수익성이 높다.

---

# 🔵 **1) EQUITY_MULTIPLIER (자기자본승수)**

### ✔ 정의

기업의 **재무 레버리지(부채 사용 정도)** 를 나타내는 지표.
자산총계가 자기자본의 몇 배인지 확인한다.

### ✔ 공식

```
자기자본승수 = 총자산 / 자기자본
```

### ✔ 해석

* 1.0~2.0 → 안정적
* 3.0↑ → 부채 활용 많음(리스크 증가)

### ✔ 예시

총자산 10조 / 자기자본 4조 = **2.5**

→ 부채를 다소 활용하는 회사

### ✔ 주의

* 승수가 높다고 “무조건 나쁘다”가 아님
* 은행/보험 등 금융업은 원래 높은 구조

---

# 🔵 **2) OCF_RATIO (영업현금흐름비율)**

### ✔ 정의

매출액 대비 영업현금흐름 비율.
→ 실제로 **현금이 잘 들어오는 비즈니스인지** 판단.

### ✔ 공식

```
영업현금흐름비율 = (영업현금흐름 / 매출액) × 100
```

### ✔ 해석

* 10%↑ → 현금창출력 우수
* 0% 근처 → 이익은 나는데 현금은 안 들어오는 구조(주의)

### ✔ 예시

매출 1조 / 영업CF 1,500억 → 15%
→ 매우 우수한 현금창출 기업

### ✔ 주의

* 재고가 늘어날 때 CF가 일시적으로 줄 수 있음
* 적자인데 CF만 높게 나오는 경우도 있음(일회성 영향)

---

# 🔵 **3) EBITDA_MARGIN (EBITDA 마진)**

### ✔ 정의

영업이익에 감가상각을 더한 매출 대비 이익률.
→ 기업의 **본질적인 영업능력**을 보여주는 지표.

### ✔ 공식

```
EBITDA 마진 = (EBITDA / 매출액) × 100
```

### ✔ 해석

* 20%↑ → 매우 뛰어난 비즈니스
* 10%~20% → 양호
* 10%↓ → 본업 경쟁력 낮음

### ✔ 예시

EBITDA 3,000억 / 매출 1조 = 30%
→ 영업 효율 최상급

### ✔ 주의

* CAPEX가 많은 기업은 실제 현금흐름과 차이 큼
* 감가상각이 크면 EBITDA가 과도하게 좋아 보일 수 있음

---

# 🔵 **4) PROFITABILITY_SCORE (수익성 종합 점수)**

### ✔ 정의

매출총이익률(GPM) + 영업이익률(OPM) + 순이익률(NPM) 등
여러 수익성 지표를 하나로 합친 **종합 점수**.

### ✔ 공식

```
수익성 점수 = GPM 점수 + OPM 점수 + NPM 점수 (가중 합산)
```

### ✔ 해석

* 높은 점수 → 전반적으로 돈 잘 버는 회사
* 낮은 점수 → 수익성 구조 취약

### ✔ 예시

* GPM 30%
* OPM 15%
* NPM 10%

→ 수익성 점수: 상위권

### ✔ 주의

* 산업별 기준이 다르므로 동종업종 비교 필수
* 일회성 비용 또는 환율 영향으로 왜곡될 수 있음

---

# 🎯 **수익성(PROFITABILITY) 팩터 요약표**

| 팩터                  | 의미        | 핵심 포인트         |
| ------------------- | --------- | -------------- |
| Equity Multiplier   | 재무 레버리지   | 부채 의존도         |
| OCF Ratio           | 현금 기반 수익성 | “현금이 진짜 들어오나?” |
| EBITDA Margin       | 본업 경쟁력    | 감가상각 제외한 순수 효율 |
| Profitability Score | 종합 수익성    | 전체적인 수익 모델의 질  |

---

# 🎯 **최종 결론**

수익성 팩터는 기업의 “돈 버는 체질”을 평가하는 데 핵심이다.

특히 다음 3개는 실전 투자에서 가장 중요하다:

* **EBITDA 마진** → 본업 경쟁력
* **OCF 비율** → 실제 현금창출능력
* **수익성 종합점수** → 전반적인 강점 평가

수익성이 높은 기업일수록 위기에도 버티고,
장기 성장에서도 **주가가 꾸준히 우상향**하는 경향이 있다.

👉 가치 + 수익성 + 성장 조합은
현실적인 퀀트 투자에서 가장 강력한 조합이다.
