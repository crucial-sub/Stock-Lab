# 퀄리티 팩터 (Quality Factors)

## 개요
퀄리티 투자는 재무 건전성과 수익성이 우수한 우량 기업을 선별하는 전략입니다.

## 주요 팩터

### 1. ROE (자기자본이익률)
- **정의**: 자기자본 대비 순이익 비율
- **공식**: `당기순이익 / 자기자본 × 100`
- **기준값**:
  - 우수: 15% 이상
  - 양호: 10-15%
  - 보통: 7-10%
- **의미**: 주주 자본의 효율성

### 2. ROA (총자산이익률)
- **정의**: 총자산 대비 순이익 비율
- **공식**: `당기순이익 / 총자산 × 100`
- **기준값**:
  - 우수: 10% 이상
  - 양호: 5-10%
- **의미**: 자산 활용 효율성

### 3. 영업이익률
- **정의**: 매출 대비 영업이익 비율
- **공식**: `영업이익 / 매출액 × 100`
- **기준값**:
  - 우수: 15% 이상
  - 양호: 10-15%
  - 보통: 5-10%
- **의미**: 본업의 수익성

### 4. 부채비율
- **정의**: 자기자본 대비 부채 비율
- **공식**: `부채총계 / 자기자본 × 100`
- **기준값**:
  - 우수: 50% 이하
  - 양호: 50-100%
  - 주의: 100% 초과
- **의미**: 재무 안정성

### 5. 유동비율
- **정의**: 유동부채 대비 유동자산 비율
- **공식**: `유동자산 / 유동부채 × 100`
- **기준값**:
  - 우수: 200% 이상
  - 양호: 150-200%
  - 주의: 100% 미만
- **의미**: 단기 지급능력

### 6. 이자보상배율
- **정의**: 영업이익 대비 이자비용 비율
- **공식**: `영업이익 / 이자비용`
- **기준값**:
  - 안전: 5배 이상
  - 양호: 3-5배
  - 위험: 1배 미만
- **의미**: 이자 지급 능력

## 전략 조합 예시

### 기본 퀄리티 전략
```json
{
  "buy_conditions": [
    {"factor": "ROE", "operator": ">", "value": 15},
    {"factor": "ROA", "operator": ">", "value": 10},
    {"factor": "부채비율", "operator": "<", "value": 100}
  ]
}
```

### 고수익성 전략
```json
{
  "buy_conditions": [
    {"factor": "ROE", "operator": ">", "value": 20},
    {"factor": "영업이익률", "operator": ">", "value": 15},
    {"factor": "순이익률", "operator": ">", "value": 10}
  ]
}
```

### 재무건전성 중심 전략
```json
{
  "buy_conditions": [
    {"factor": "부채비율", "operator": "<", "value": 50},
    {"factor": "유동비율", "operator": ">", "value": 150},
    {"factor": "ROE", "operator": ">", "value": 12}
  ]
}
```

### 종합 퀄리티 전략
```json
{
  "buy_conditions": [
    {"factor": "ROE", "operator": ">", "value": 15},
    {"factor": "영업이익률", "operator": ">", "value": 10},

    {"factor": "유동비율", "operator": ">", "value": 130}
  ]

```

## ROE 분해 분석 (듀퐁 분석)

```
ROE = 순이익률 × 자산회전율 × 레버리지
    = (순이익/매출) × (매출/자산) × (자산/자기자본)
```

### 분석 포인트
1. **순이익률 중심**: 고마진 비즈니스 (소프트웨어, 제약)
2. **자산회전율 중심**: 효율적 운영 (유통, 소매)
3. **레버리지 중심**: 적정 부채 활용 (금융)

## 업종별 기준값

### 제조업
- ROE: 10-15%
- 영업이익률: 8-12%
- 부채비율: 80-120%

### IT/소프트웨어
- ROE: 15-25%
- 영업이익률: 15-30%
- 부채비율: 30-60%

### 금융
- ROE: 8-12%
- ROA: 0.5-1.5%
- 부채비율: 500-800% (업종 특성)

### 유통/소매
- ROE: 10-15%
- 영업이익률: 3-7%
- 부채비율: 60-100%

## 퀄리티 스코어 계산 예시

```python
quality_score = (
    (ROE >= 15) * 2 +          # 수익성
    (ROA >= 10) * 1 +          # 자산효율
    (영업이익률 >= 10) * 1 +    # 본업수익
    (부채비율 <= 100) * 2 +     # 재무건전
    (유동비율 >= 150) * 1       # 단기안정
)
# 최대 7점, 5점 이상 우량
```

## 주의사항

1. **회계 조작**:
   - 매출 부풀리기
   - 비용 이연
   - 감가상각 조정

2. **업종 특성**:
   - 자본집약적 산업: ROE/ROA 낮음
   - 금융업: 부채비율 높음
   - 유통업: 마진 낮음

3. **시기 고려**:
   - 경기 사이클
   - 계절성
   - 일회성 손익

4. **지속가능성**:
   - 3-5년 추세 확인
   - ROE 개선 방식 분석
   - 경쟁우위 지속성

## 검증 체크리스트

- [ ] ROE 15% 이상 3년 유지
- [ ] 영업이익률 개선 추세
- [ ] 부채비율 안정적 관리
- [ ] 현금흐름 양호

- [ ] 업종 내 상위권 수익성
