📄 퀄리티 팩터 (Quality Factor) — 초보자용 완전 정리 (10개)

퀄리티 팩터는 “얼마나 좋은 회사인가?” 를 판단하는 전략이에요.
주로 수익성, 효율성, 건전성, 회계 품질을 평가합니다.

💡 핵심: “싸냐?”가 아니라 **“좋은 회사냐?”**를 보는 팩터

# 퀄리티 팩터 핵심 지표 10개
## 1. ROE (자기자본수익률)
✔ 한 줄 정의

주주가 투자한 돈으로 얼마나 이익을 잘 내는지 보는 지표.

✔ 공식
ROE = 당기순이익 / 자기자본 × 100

✔ 해석 포인트

ROE 높을수록 우량기업

10% 이상이면 좋은 편, 15% 이상이면 매우 우수

✔ 초보자 예시

A 기업 ROE = 14% → 돈을 잘 버는 회사
B 기업 ROE = 4% → 자본 효율 낮음

✔ 주의사항

일시적 이익 증가로 ROE가 급등할 수 있음

부채가 너무 많아도 ROE가 인위적으로 상승함

## 2. ROA (총자산수익률)
✔ 정의

기업 전체 자산을 얼마나 효율적으로 운영하는지 보는 지표.

✔ 공식
ROA = 당기순이익 / 총자산 × 100

✔ 해석

ROA가 높을수록 “전체 자산 대비 이익률”이 좋음

제조업은 ROA가 낮고, IT기업은 높을 수 있음

✔ 예시

ROA 8% → 매우 우수
ROA 2% → 자산 많이 쓰고 이익 적게 냄

✔ 주의

대규모 설비기업은 ROA가 낮아도 정상일 수 있음

## 3. ROIC (투하자본수익률)
✔ 정의

“투자한 자본(부채+자기자본) 대비 얼마나 높은 수익을 내는가”

✔ 공식
ROIC = NOPAT / (투하자본) × 100

✔ 해석

ROIC > 10% → 고효율 기업

ROIC는 ROE보다 왜곡이 적어 더 정교함

✔ 예시

ROIC 12% → 투자 대비 우수한 수익
ROIC 3% → 비효율적 사업 구조

✔ 주의

NOPAT 추정 방식에 따라 계산이 조금 달라질 수 있음

## 4. GPM (매출총이익률)
✔ 정의

제품을 팔아서 얼마나 남기는지 보는 지표.

✔ 공식
GPM = 매출총이익 / 매출액 × 100

✔ 해석

높을수록 본업 경쟁력 좋음

브랜드 강한 기업(GPM 높음)

✔ 예시

애플 GPM: 40~45%
일반 제조업: 10~20%

✔ 주의

원재료 가격 변동에 민감하게 반응

## 5. OPM (영업이익률)
✔ 정의

본업으로 벌어들이는 순수 이익률.

✔ 공식
OPM = 영업이익 / 매출액 × 100

✔ 해석

OPM 높을수록 비용 통제 잘함

10% 이상이면 우량기업

✔ 예시

OPM 15% → 비용관리 뛰어남
OPM 3% → 본업 경쟁력 떨어짐

✔ 주의

일회성 비용 포함 여부 확인 필요

## 6. NPM (순이익률)
✔ 정의

매출 대비 최종적으로 남는 이익.

✔ 공식
NPM = 순이익 / 매출액 × 100

✔ 해석

높을수록 경영 효율 및 재무건전성 좋음

기술기업은 높고, 유통업은 낮음

✔ 예시

NPM 12% → 매우 우수
NPM 1% → 상당히 낮음

✔ 주의

세금, 영업외손익에 따라 수익률 크게 변함

## 7. Asset Turnover (자산회전율)
✔ 정의

기업이 가진 자산을 얼마나 빠르게 매출로 바꾸는지.

✔ 공식
자산회전율 = 매출액 / 총자산

✔ 해석

높을수록 효율적인 회사

유통업은 매우 높고, 제조업은 낮음

✔ 예시

A 기업 1.2 → 자산 효율적
B 기업 0.4 → 자산 비효율

✔ 주의

산업별 기준이 큰 차이가 있음

## 8. Inventory Turnover (재고자산회전율)
✔ 정의

재고를 얼마나 빨리 소진/판매하는지.

✔ 공식
재고자산회전율 = 매출원가 / 재고자산

✔ 해석

높을수록 재고가 잘 팔리고 운영이 효율적

재고 쌓이는 기업은 위험 신호

✔ 예시

재고회전율 10 → 빠른 판매
재고회전율 2 → 재고 누적 위험

✔ 주의

계절성 산업은 회전율이 들쑥날쑥함

## 9. Quality Score (품질점수, Piotroski F-score)
✔ 정의

재무 건전성, 수익성, 효율성을 종합한 점수(0~9점).

✔ 공식
9개 재무항목 종합 점수

✔ 해석

7~9점 → 매우 우량

0~3점 → 매우 위험

✔ 예시

F-score 8점 → 재무구조 강함

✔ 주의

점수는 연도별로 크게 변할 수 있음

산업별 비교 필요

## 10. Accruals Ratio (발생액 비율)
✔ 정의

회계상 이익과 실제 현금흐름의 차이를 보는 지표.

✔ 공식
Accruals = (순이익 - 영업현금흐름) / 총자산

✔ 해석

낮을수록 품질 높은 이익

높으면 회계적으로 “이익 부풀리기” 가능성

✔ 예시

Accruals 0.05 → 정상
Accruals 0.25 → 위험 신호

✔ 주의

일회성 비용·이익에 의해 왜곡 가능

성장기업(재고 증가)에서는 자연스레 높아질 수도 있음

# 🎯 퀄리티 팩터 요약표

| 지표                 | 의미           | 기준          |
| ------------------ | ------------ | ----------- |
| ROE                | 주주 자본 대비 수익률 | 10%↑ 우량     |
| ROA                | 자산 대비 수익률    | 높을수록 좋음     |
| ROIC               | 투하자본 대비 수익률  | 10%↑ 우수     |
| GPM                | 매출총이익률       | 높을수록 본업 경쟁력 |
| OPM                | 영업이익률        | 10%↑ 우량     |
| NPM                | 순이익률         | 산업마다 편차 큼   |
| Asset Turnover     | 자산 효율        | 높을수록 효율     |
| Inventory Turnover | 재고 소진 효율     | 높을수록 좋음     |
| Quality Score      | 종합 품질(0~9)   | 7점↑ 우량      |
| Accruals Ratio     | 이익 품질        | 낮을수록 정상     |


# 🎯 최종 결론

퀄리티 팩터는 **“좋은 회사인가?”**를 판단하는 최고의 기준입니다.
특히 ROE·ROIC·OPM·NPM은 기업 체질을 직접 보여주기 때문에
성장주·우량주를 고르는 데 필수적인 지표입니다.

✔ 수익성 좋고
✔ 효율적으로 운영되고
✔ 재무 건전하고
✔ 이익 품질까지 좋은 기업은
시장에서 고평가되어도 오랫동안 강한 성과를 보이는 경향이 있습니다.

퀄리티 팩터는 가치/모멘텀과 함께 조합하면
가장 안정적이면서 성과가 강한 전략이 됩니다.
