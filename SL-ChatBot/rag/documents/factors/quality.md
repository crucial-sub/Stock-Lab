# 📄 **퀄리티 팩터 (Quality Factor) **

퀄리티 팩터는 기업이 **얼마나 좋은 회사인지** 평가하는 팩터다.
수익성, 효율성, 건전성, 회계 품질 등을 기반으로
“이 회사의 체질이 좋은가?”를 판단한다.

> 💡 핵심: “싸냐?”가 아니라 **“퀄리티가 좋냐?”**를 본다.

---

# 1) **ROE (자기자본수익률)**

### ✔ 정의

주주가 투자한 자기자본으로 얼마나 효율적으로 이익을 냈는지 보여주는 대표 수익성 지표.

### ✔ 공식

```
ROE = 당기순이익 / 자기자본 × 100
```

### ✔ 해석

* 10% 이상 → 우량
* 15% 이상 → 매우 우수
* 높을수록 자본 효율이 좋음

### ✔ 예시

A기업: 14% → 자본 효율 우수
B기업: 4% → 자본 효율 낮음

### ✔ 주의

* 부채가 많으면 ROE가 인위적으로 높아질 수 있음
* 일회성 이익에도 쉽게 변함

---

# 2) **ROA (총자산수익률)**

### ✔ 정의

기업 전체 자산을 사용해 얼마의 이익을 냈는지 나타내는 지표.

### ✔ 공식

```
ROA = 당기순이익 / 총자산 × 100
```

### ✔ 해석

* 높을수록 전체 자산 효율성이 좋음
* IT·서비스업 > 제조업

### ✔ 예시

ROA 8% → 우수
ROA 2% → 자산을 비효율적으로 사용

### ✔ 주의

* 설비 산업은 ROA 낮아도 정상
* 업종 비교 필수

---

# 3) **ROIC (투하자본수익률)**

### ✔ 정의

부채 + 자기자본(투하자본) 전체 대비 얼마나 높은 수익을 내는지.

### ✔ 공식

```
ROIC = NOPAT / 투하자본 × 100
```

### ✔ 해석

* 10% 이상 → 고퀄리티 기업
* ROE보다 회계 왜곡이 적어 더 정교함

### ✔ 예시

ROIC 12% → 투자 효율 우수
ROIC 3% → 비효율적 사업 구조

### ✔ 주의

* NOPAT 추정 방식 차이 존재
* 산업마다 평균치 크게 다름

---

# 4) **GPM (매출총이익률)**

### ✔ 정의

제품·서비스를 판매해 남기는 기본 이익률.

### ✔ 공식

```
GPM = 매출총이익 / 매출액 × 100
```

### ✔ 해석

* 높을수록 본업 경쟁력 강함
* 브랜드 기업은 GPM 높게 유지

### ✔ 예시

애플: 40~45%
일반 제조업: 10~20%

### ✔ 주의

* 원재료 가격에 민감
* 경쟁 심해지면 GPM 하락

---

# 5) **OPM (영업이익률)**

### ✔ 정의

본업에서 남기는 순수 이익률.

### ✔ 공식

```
OPM = 영업이익 / 매출액 × 100
```

### ✔ 해석

* 10% 이상 → 우량
* 비용 통제 능력을 직접 반영

### ✔ 예시

OPM 15% → 매우 우수
OPM 3% → 경쟁력 약함

### ✔ 주의

* 일회성 비용 포함 여부 확인
* 경기 영향 커서 변동성 높을 수 있음

---

# 6) **NPM (순이익률)**

### ✔ 정의

매출 대비 최종적으로 남는 순이익 비율.

### ✔ 공식

```
NPM = 순이익 / 매출액 × 100
```

### ✔ 해석

* 높을수록 경영 효율 및 재무 건전성 우수
* IT/플랫폼 > 제조업 > 유통업 순

### ✔ 예시

NPM 12% → 매우 우수
NPM 1% → 매우 낮음

### ✔ 주의

* 세금, 영업외손익에 따라 쉽게 흔들림

---

# 7) **ASSET_TURNOVER (자산회전율)**

### ✔ 정의

기업이 보유한 자산을 얼마나 효율적으로 매출로 전환하는지.

### ✔ 공식

```
자산회전율 = 매출액 / 총자산
```

### ✔ 해석

* 높을수록 효율적
* 유통업 > IT > 제조업

### ✔ 예시

1.2 → 자산 효율 우수
0.4 → 자산 비효율

### ✔ 주의

* 산업별 편차 매우 큼
* 제조업은 낮아도 정상

---

# 8) **INVENTORY_TURNOVER (재고자산회전율)**

### ✔ 정의

재고를 얼마나 빨리 판매·소진하는지.

### ✔ 공식

```
재고회전율 = 매출원가 / 재고자산
```

### ✔ 해석

* 높을수록 재고 관리 우수
* 낮으면 재고 누적 → 리스크 증가

### ✔ 예시

회전율 10 → 빠른 소진
회전율 2 → 재고 누적 위험

### ✔ 주의

* 계절성 산업은 진폭 큼
* 재고 평가 방식에 따라 왜곡 가능

---

# 9) **QUALITY_SCORE (피오트로스키 F-score)**

### ✔ 정의

9가지 재무 항목을 기반으로 기업의 종합 재무 품질을 평가하는 점수 (0~9점).

### ✔ 공식

```
수익성 + 건전성 + 효율성 항목 점수(총 9점)
```

### ✔ 해석

* 7~9점 → 매우 우량
* 0~3점 → 고위험

### ✔ 예시

품질 점수 8점 → 재무 체질 매우 우수

### ✔ 주의

* 산업별 비교 필요
* 점수 변동 폭 클 수 있음

---

# 10) **ACCRUALS_RATIO (발생액 비율)**

### ✔ 정의

회계상 이익과 실제 현금흐름의 차이를 측정하는 지표.

### ✔ 공식

```
Accruals = (순이익 - 영업현금흐름) / 총자산
```

### ✔ 해석

* 낮을수록 이익 품질 우수
* 높으면 “회계상 이익 부풀리기” 가능성

### ✔ 예시

0.05 → 정상
0.25 → 품질 낮음 (경고)

### ✔ 주의

* 일회성 비용/이익에 의해 왜곡 가능
* 성장기업은 자연스럽게 높아질 수도 있음

# 🎯 **퀄리티 팩터 최종 요약**

| 지표                     | 의미        | 기준          |
| ---------------------- | --------- | ----------- |
| **ROE**                | 자본 대비 수익성 | 10%↑ 우량     |
| **ROA**                | 자산 대비 수익성 | 높을수록 우수     |
| **ROIC**               | 투자 대비 수익성 | 10%↑ 우수     |
| **GPM**                | 매출총이익률    | 높을수록 본업 경쟁력 |
| **OPM**                | 영업이익률     | 10%↑ 우량     |
| **NPM**                | 순이익률      | 산업별 편차 큼    |
| **Asset Turnover**     | 자산 효율성    | 높을수록 효율적    |
| **Inventory Turnover** | 재고 효율성    | 높을수록 우수     |
| **Quality Score**      | 종합 품질 0~9 | 7↑ 우량       |
| **Accruals Ratio**     | 이익 품질     | 낮을수록 정상     |

# 🎯 **최종 결론**

퀄리티 팩터는 **“좋은 회사인가?”**를 판단하는 가장 강력한 기준이다.

특히 아래 4개는 기업 체질을 직접적으로 보여주는 핵심 지표다:

* **ROE** (주주 자본 효율)
* **ROIC** (투하자본 효율성)
* **OPM** (본업 경쟁력)
* **NPM** (최종 수익성)

이 네 가지가 좋은 기업은
✔ 수익성이 높고
✔ 효율적으로 운영되고
✔ 재무적으로 건전하며
✔ 이익 품질도 좋기 때문에

장기적으로 **고평가 상태에서도 시장 대비 높은 초과수익을 낼 가능성이 높다.**

퀄리티 팩터는 **가치(Value)·모멘텀(Momentum)** 과 조합되면
가장 강력하고 안정적인 투자 전략이 된다.
