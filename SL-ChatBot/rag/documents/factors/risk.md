# 📄 **리스크 팩터 (Risk Factor) – 4개 **

---

# # 1) **SHARPE_RATIO (샤프 비율)**

### ✔ 정의

투자 위험(변동성)을 감안했을 때,
**얼마나 효율적으로 수익을 냈는지** 보여주는 지표.

### ✔ 공식

```
Sharpe = (수익률 - 무위험수익률) / 변동성
```

### ✔ 해석

* **1.0 이상** → 우수
* **0.5~1.0** → 보통
* **0.5 이하** → 위험 대비 보상이 낮음

### ✔ 예시

수익률 10%, 변동성 5%, 무위험 2%
→ (10-2)/5 = **1.6 (아주 우수)**

### ✔ 주의

* 하락만 따로 보지 못함
* 변동성이 낮으면 과대평가 가능

---

# # 2) **SORTINO_RATIO (소르티노 비율)**

### ✔ 정의

샤프 비율에서 단점을 보완해
**하방 위험(떨어질 때만)**을 기준으로 평가하는 지표.

### ✔ 공식

```
Sortino = (수익률 - 무위험수익률) / 하락변동성
```

### ✔ 해석

* **샤프 > 소르티노** → 일반적
* **소르티노가 높다** → 하락 구간에서 잘 버틴 종목

### ✔ 예시

수익률 12%, 무위험 2%, 하락변동성 4%
Sortino = 2.5 → 매우 우수

### ✔ 주의

* 상승할 때의 변동성은 무시
* 너무 단기에 계산하면 왜곡됨

---

# # 3) **DOWNSIDE_VOLATILITY (하방 변동성)**

### ✔ 정의

종목이 **하락했을 때의 변동성(리스크)**만 분리해서 측정한 지표.

### ✔ 공식

```
음수 수익률들의 표준편차
```

### ✔ 해석

* 낮을수록 안전
* 높을수록 하락 구간에서 흔들림 심함

### ✔ 예시

하방 변동성 3% → 안정적
하방 변동성 15% → 매우 위험

### ✔ 주의

* 전체 변동성보다 중요한 경우가 많음
* 시장폭락 구간에 민감

---

# # 4) **MAX_DRAWDOWN (최대 낙폭, MDD)**

### ✔ 정의

투자 기간 중 **가장 깊었던 하락폭**
= 최고점 대비 얼마나 빠졌는가?

### ✔ 공식

```
MDD = (저점 - 고점) / 고점
```

### ✔ 해석

* -10% ~ -20% → 정상적
* -30%↓ → 장기 손실 가능성
* -50%↓ → 매우 위험 (회복 어려움)

### ✔ 예시

고점 100, 저점 70
MDD = -30% → 큰 폭 하락 경험

### ✔ 주의

* 회복에 매우 긴 시간 필요
* 모멘텀·성장주는 MDD가 크게 나오는 경향

# 🎯 **최종 요약**

| 팩터                      | 의미          | 핵심 포인트         |
| ----------------------- | ----------- | -------------- |
| **Sharpe Ratio**        | 변동성 대비 수익   | 전체 위험 기준       |
| **Sortino Ratio**       | 하락 위험 대비 수익 | 하락만 평가 (더 정교함) |
| **Downside Volatility** | 하락 시 흔들림    | 낮을수록 안정적       |
| **Max Drawdown**        | 역사상 최악 낙폭   | 회복 가능성 평가 핵심   |

---

# 🎯 **최종 결론**

이 4가지 RISK 팩터는
**“이 종목이 떨어질 때 얼마나 위험한가?”**,
**“수익을 얼마나 효율적으로 냈나?”**를 측정하는 핵심 지표들이다.

특히:

* 📌 **MDD(최대 낙폭)** → 실제 투자자 손실 체감과 가장 가까움
* 📌 **Sortino Ratio** → 하락 위험 중심으로 더 정교함
* 📌 **Sharpe Ratio** → 전체 위험 대비 효율성
* 📌 **Downside Volatility** → 하락 때 얼마나 흔들리는지

이 네 가지를 함께 보면
👉 **“위험 대비 괜찮은 종목인가?”**를 정확히 판단할 수 있다.
