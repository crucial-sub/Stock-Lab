
# 📘 **재무 안정성 팩터 (Stability Factor) — 초보자용 완전 정리 (10개)**

재무 안정성(STABILITY)은 기업이 **위기 상황에서도 버틸 힘이 있는가?**
부채, 유동성, 이자갚는 능력, 파산 위험 등을 통해 기업의 **안전마진(Safety Margin)** 을 평가한다.

> 💡 핵심: “얼마나 위험하냐?”를 보는 팩터
> 불황, 금리 인상기, 변동성 장세에서 특히 중요!

---

# 1) **DEBT_TO_EQUITY (부채비율)**

### ✔ 정의

자기자본 대비 부채가 얼마나 많은지 측정하는 대표 부채 지표.

### ✔ 공식

```
부채총계 ÷ 자기자본 × 100
```

### ✔ 해석

* **100% 이하 → 안정적**
* **200% 이상 → 재무 위험 증가**
* 낮을수록 기업이 위기 시 버틸 힘이 강함

### ✔ 예시

부채 2조 / 자기자본 1조 = **200%**

### ✔ 주의

* 금융업은 특성상 높아도 정상
* 적자 기업은 자기자본 감소로 비율 왜곡 가능

---

# 2) **DEBT_RATIO (총부채비율)**

### ✔ 정의

총자산 중 부채가 차지하는 비율.

### ✔ 공식

```
부채총계 ÷ 총자산 × 100
```

### ✔ 해석

* **30~50% → 일반적 안정 구간**
* 낮을수록 안전하지만 너무 낮으면 성장성 부족 가능

### ✔ 예시

부채 4조 / 자산 10조 = **40%**

### ✔ 주의

* 산업별 평균을 꼭 비교해야 함

---

# 3) **CURRENT_RATIO (유동비율)**

### ✔ 정의

1년 내 갚아야 하는 부채를 유동자산으로 얼마나 커버 가능한지.

### ✔ 공식

```
유동자산 ÷ 유동부채 × 100
```

### ✔ 해석

* **100% 이상 → 단기 상환능력 안전**
* 200% 이상이면 여유 매우 많음

### ✔ 예시

유동자산 3000억 / 유동부채 1000억 = **300%**

### ✔ 주의

* 재고 비중 높은 기업은 유동비율이 높아도 안전하지 않을 수 있음

---

# 4) **QUICK_RATIO (당좌비율)**

### ✔ 정의

재고를 제외한 즉시 현금화 가능한 자산의 단기 상환능력.

### ✔ 공식

```
(유동자산 - 재고자산) ÷ 유동부채 × 100
```

### ✔ 해석

* **100% 이상이면 매우 안전**
* 제조업은 70~80%도 정상

### ✔ 예시

(3000억 - 1200억) / 1000억 = **180%**

### ✔ 주의

* 업종별 차이 매우 큼
* 계절·재고 비중에 따라 변동 가능

---

# 5) **INTEREST_COVERAGE (이자보상배율)**

### ✔ 정의

영업이익으로 이자비용을 얼마나 감당하는지.

### ✔ 공식

```
EBIT ÷ 이자비용
```

### ✔ 해석

* **2 이상 → 안정적**
* **1 미만 → 이자도 못 갚는 위험 기업**

### ✔ 예시

EBIT 500억 / 이자비용 200억 = **2.5**

### ✔ 주의

* 단기 실적 변동·일회성 비용에 민감

---

# 6) **ALTMAN_Z_SCORE (알트만 Z-Score)**

### ✔ 정의

기업의 파산 가능성을 예측하는 종합 위험 지표.

### ✔ 공식 (제조업 기준)

```
1.2×운전자본/자산
+ 1.4×유보이익/자산
+ 3.3×EBIT/자산
+ 0.6×시가총액/부채총계
+ 1.0×매출액/자산
```

### ✔ 해석

* **3.0 이상 → 안전**
* **1.8 이하 → 파산 위험 구간**

### ✔ 예시

Z = **3.2** → 매우 안정적

### ✔ 주의

* 금융업/서비스업엔 정확도 떨어짐
* 장부가치 데이터에 영향 받음

---

# 7) **BETA (베타)**

### ✔ 정의

시장의 변동성 대비 개별 주가 변동성이 얼마나 큰지 나타내는 지표.

### ✔ 공식

```
개별 수익률 변동성 ÷ 시장 수익률 변동성
```

(회귀분석으로 계산하는 것이 일반적)

### ✔ 해석

* **1 초과 → 시장보다 변동성 큼 (공격적)**
* **1 미만 → 시장보다 안정적 (방어적)**

### ✔ 예시

β = **1.5** → 시장보다 1.5배 움직임

### ✔ 주의

* 성장주·테크주는 베타가 높게 형성됨
* 시장 충격에 민감

---

# 8) **EARNINGS_QUALITY (이익 품질)**

### ✔ 정의

순이익이 실제 현금흐름(OCF)으로 얼마나 뒷받침되는지.

### ✔ 공식

```
영업현금흐름(OCF) ÷ 당기순이익
```

### ✔ 해석

* **1 이상 → 양질의 이익**
* **1 미만 → 회계상 이익이 부풀려졌을 가능성**

### ✔ 예시

OCF 800억 / 순이익 600억 = **1.33**

### ✔ 주의

* 일회성 이익·현금 흐름 변동에 주의
* 성장기업은 자연스럽게 낮아질 수도 있음

---

# 9) **CASH_RATIO (현금비율)**

### ✔ 정의

유동부채를 ‘현금만’으로 얼마나 커버할 수 있는지.

### ✔ 공식

```
현금 및 현금성 자산 ÷ 유동부채 × 100
```

### ✔ 해석

* **20% 이상 → 안정**
* **50% 이상 → 매우 안전**

### ✔ 예시

1000억 / 2500억 = **40%**

### ✔ 주의

* 너무 높으면 성장성 저하 가능(현금 쌓아두기)

---

# 10) **STABILITY_SCORE (종합 안정성 점수)**

### ✔ 정의

부채비율·유동성·이자보상배율 등 복수 데이터로 만든 기업 안정성 종합 점수.

### ✔ 공식

```
각 안정성 지표의 가중합 점수
(모델에 따라 구성항목 상이)
```

### ✔ 해석

* 점수 높을수록 재무가 안정적
* 단일 지표보다 신뢰도 높음

### ✔ 예시

Stability Score = **78점** → 안정적

### ✔ 주의

* 재무모델마다 공식이 다름
* 업종 특성 반영 필요

# 🎯 **재무 안정성(STABILITY) — 최종 요약**

| 지표                    | 의미         | 투자 포인트     |
| --------------------- | ---------- | ---------- |
| **Debt to Equity**    | 자기자본 대비 부채 | 부채 의존도     |
| **Debt Ratio**        | 자산 대비 부채   | 재무 구조 안전성  |
| **Current Ratio**     | 단기 상환능력    | 100%↑ 안정   |
| **Quick Ratio**       | 즉시 상환능력    | 재고 제외      |
| **Interest Coverage** | 이자 갚는 능력   | 2배↑ 우량     |
| **Altman Z-score**    | 파산 위험      | 3↑ 안전      |
| **Beta**              | 변동성 위험     | 1↓ 안정적     |
| **Earnings Quality**  | 현금 기반 이익   | 1↑ 양질      |
| **Cash Ratio**        | 현금 커버력     | 20~50% 안정  |
| **Stability Score**   | 종합 안전성     | 점수 높을수록 좋음 |

# 🎯 **최종 결론**

STABILITY 팩터는
**“이 기업이 위기에서도 절대 망하지 않나?”**
를 판단하는 가장 중요한 안전성 지표이다.

특히:

* **부채비율(D/E)**
* **이자보상배율(Interest Coverage)**
* **Altman Z-score**
* **현금비율(Cash Ratio)**
* **이익품질(Earnings Quality)**

이 다섯 가지는 기업의 **붕괴 위험, 단기 상환능력, 재무 체력**을 직관적으로 보여준다.

> 안정성이 높은 기업은,
> **폭락장에서 덜 떨어지고 회복도 빠르며 장기 복리 효과가 훨씬 크다.**
