# 📊 **TECHNICAL 팩터 (Technical Factor) — 초보자용 완전 정리 (28개)**

가격·거래량·변동성 기반으로 **현재 추세 / 모멘텀 / 과열 / 침체 / 전환 신호**를 판단하는 지표들.

> 💡 핵심: “지금 이 종목의 움직임이 어떤가?”

---

# 🔵 **1. RSI_14 / RSI (상대강도지수)**

### ✔ 정의

14일 동안 상승폭과 하락폭을 비교해 **과열/침체**를 판단.

### ✔ 공식

```
RSI = 100 - (100 / (1 + RS))
RS = 평균 상승폭 / 평균 하락폭
```

### ✔ 해석

* 70↑ → 과열
* 30↓ → 과매도

### ✔ 예시

RSI 78 → 단기 과열
RSI 25 → 단기 저점 가능

### ✔ 주의

강한 상승장에서는 RSI가 계속 70 이상일 수 있음.

---

# 🔵 **2. BOLLINGER_POSITION (볼린저밴드 위치)**

### ✔ 정의

현재가가 볼린저밴드 상·하단 중 어디에 있는지.

### ✔ 공식

```
(현재가 - 하단밴드) / (상단밴드 - 하단밴드)
```

### ✔ 해석

* 0.8↑ ⇒ 상단 근접(과열)
* 0.2↓ ⇒ 하단 근접(침체)

### ✔ 주의

박스권(횡보장)에서는 의미가 약해짐.

---

# 🔵 **3. MACD_SIGNAL (MACD 시그널)**

### ✔ 정의

MACD의 전환점을 나타내는 지표.
“골든/데드 크로스” 판단에 사용.

### ✔ 공식

```
MACD = EMA12 - EMA26
Signal = EMA9(MACD)
```

### ✔ 해석

* MACD > Signal → 상승 전환
* MACD < Signal → 하락 전환

### ✔ 예시

MACD가 시그널을 상향 돌파 → 매수 신호

### ✔ 주의

횡보장에서는 속임수 신호가 매우 많음.

---

# 🔵 **4. STOCHASTIC_14 (스토캐스틱)**

### ✔ 정의

최근 14일 고가/저가 대비 현재 가격이 어디인지.

### ✔ 공식

```
%K = (현재가 - 최저가) / (최고가 - 최저가)
```

### ✔ 해석

* 80↑ → 과열
* 20↓ → 과매도

---

# 🔵 **5. VOLUME_ROC (거래량 변화율)**

### ✔ 정의

거래량이 전날 대비 얼마나 증가/감소했는지.

### ✔ 공식

```
(현재 거래량 - 과거 거래량) / 과거 거래량 × 100
```

### ✔ 해석

* 거래량 폭발 → 매수/매도 강세
* 거래량 감소 → 관심 감소

### ✔ 주의

뉴스·테마 영향에 민감.

---

# 🔵 **6. PRICE_POSITION (가격 위치)**

### ✔ 정의

현재 가격이 **52주 범위** 내에서 어디에 있는지.

### ✔ 해석

* 상단부 근접 → 강한 종목
* 하단부 근접 → 저평가 구간 가능

---

# 🔵 **7. BOLLINGER_WIDTH (볼린저 폭)**

### ✔ 정의

볼린저밴드 상단~하단 폭 = 변동성 크기.

### ✔ 해석

* 폭↑ → 변동성 증가
* 폭↓ → 추세 시작 전 조용한 구간

---

# 🔵 **8. MACD**

### ✔ 정의

단기 추세와 장기 추세의 차이.

### ✔ 해석

* MACD > 0 → 상승 추세
* MACD < 0 → 하락 추세

---

# 🔵 **9. MACD_HISTOGRAM**

### ✔ 정의

MACD와 시그널선 차이(모멘텀 강도).

### ✔ 해석

* 증가 → 상승 모멘텀 강화
* 감소 → 하락 모멘텀 강화

---

# 🔵 **10. ATR (평균진폭)**

### ✔ 정의

일일 변동폭(TR)을 평균낸 값.

### ✔ 해석

* ATR↑ → 변동성 커짐
* ATR↓ → 추세 전 고요 단계

---

# 🔵 **11. VWAP**

### ✔ 정의

거래량 기준의 하루 평균 가격.

### ✔ 해석

* 가격 > VWAP → 매수 우위
* 가격 < VWAP → 매도 우위

---

# 🔵 **12. OBV (On Balance Volume)**

### ✔ 정의

거래량을 방향성 있게 누적한 지표.

### ✔ 해석

* 상승 → 매수세 유입
* 하락 → 매도세 압도

---

# 🔵 **13. MFI (자금흐름지수)**

### ✔ 정의

가격 + 거래량 → 자금 유입/유출 판단.

### ✔ 해석

* 80↑ → 과열
* 20↓ → 과매도

---

# 🔵 **14. WILLIAMS_R (%R)**

### ✔ 정의

현재 가격이 최근 고저 범위 중 얼마나 낮은지.

### ✔ 해석

* -20↑ → 과열
* -80↓ → 침체

---

# 🔵 **15. ADX (추세 강도)**

### ✔ 정의

현재 시장이 “추세”인지 “횡보”인지 판단.

### ✔ 해석

* 25↑ → 강한 추세
* 20↓ → 추세 없음

---

# 🔵 **16. CCI**

### ✔ 정의

가격이 평균에서 얼마나 이탈했는지.

### ✔ 해석

* +100↑ → 과열
* -100↓ → 침체

---

# 🔵 **17. AROON_UP / 18. AROON_DOWN**

### ✔ 정의

최근 고점·저점 형성 시점을 기준으로 추세 방향 파악.

### ✔ 해석

* Aroon UP 높음 → 상승 추세
* Aroon DOWN 높음 → 하락 추세

---

# 🔵 **19. CMF (Chaikin Money Flow)**

### ✔ 정의

가격·거래량 기반 자금 흐름.

### ✔ 해석

* 양수 → 매수 우위
* 음수 → 매도 우위

---

# 🔵 **20. ULTIMATE_OSCILLATOR**

### ✔ 정의

3개의 기간 모멘텀을 결합한 보조지표.

### ✔ 해석

* 70↑ → 과열
* 30↓ → 과매도

---

# 🔵 **21. TRIX**

### ✔ 정의

3중 EMA 기반 장기 모멘텀 지표.

### ✔ 해석

* 0↑ → 상승 모멘텀
* 0↓ → 하락 모멘텀

---

# 🔵 **22~26. MA(이동평균선) – MA_5, MA_20, MA_60, MA_120, MA_250**

### ✔ 정의

최근 N일 종가 평균.

### ✔ 해석

* 가격 > MA → 상승 추세
* 가격 < MA → 하락 추세

### ✔ 핵심

* MA20 = 가장 실전에서 많이 쓰는 기준선
* MA60·MA120 = 중기/장기 추세

---

# 🔵 **27. PRICE_VS_MA20**

### ✔ 정의

현재 가격이 20일선 대비 얼마나 위/아래인가.

### ✔ 공식

```
(현재가 - MA20) / MA20 × 100
```

### ✔ 해석

* +5%↑ → 단기 과열
* -5%↓ → 단기 침체

# 🎯 **TECHNICAL 팩터 최종 요약**

| 그룹        | 대표 지표                              | 의미        |
| --------- | ---------------------------------- | --------- |
| **모멘텀**   | RSI, MACD, TRIX, STOCH             | 상승/하락 강도  |
| **변동성**   | ATR, 볼린저폭                          | 가격 진폭     |
| **추세**    | ADX, MA                            | 트렌드 방향/강도 |
| **수급**    | OBV, CMF, MFI, VWAP                | 매수·매도 힘   |
| **위치 기반** | Bollinger Position, Price Position | 가격 위치     |

---

# 🎯 **최종 결론**

기술적 팩터는 단기 타이밍을 볼 때 필수이며
다른 팩터(가치/성장/퀄리티)와 조합하면 가장 강력해진다.

특히 아래 5개는 실전 필수:

* **RSI (과열/침체 판단)**
* **MACD Signal (전환 시그널)**
* **MA20 (기준 추세선)**
* **ATR (변동성 크기)**
* **OBV (수급 강도)**

→ “지금 사도 되는가?” “추세가 강한가?” 직관적으로 판단 가능.
