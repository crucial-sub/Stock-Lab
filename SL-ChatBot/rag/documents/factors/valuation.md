# 평가 팩터 (Valuation Factor)

📌 개념

평가 팩터는 **"기업의 현재 주가가 합리적 가치를 반영하고 있는지"**를 종합적으로 평가하는 전략이에요.
VALUE 팩터와 달리 여러 지표를 종합해서 절대적 가치를 판단합니다.

💡 핵심: "현재 가격이 공정한가?"

# 평가 팩터 주요 지표

## 1. 정상주가 추정 (Target Price / Fair Value)
✔ 정의

분석가들이 주식의 적정 가치를 계산한 목표주가

✔ 공식

DCF(현금흐름할인법), 상대평가 등으로 산출

✔ 해석

현재가 < 목표가 → 저평가
현재가 > 목표가 → 고평가

✔ 주의

분석가별로 추정이 다를 수 있음

## 2. 상대평가 (Relative Valuation)
✔ 정의

동일 산업 기업들과 비교해서 가치 판단

✔ 예시

같은 업종 회사들의 평균 PER이 15배일 때
해당 회사 PER 12배 → 저평가

✔ 주의

산업 특성이 다르면 비교 의미 낮음

## 3. 절대평가 (Absolute Valuation)
✔ 정의

기업 자체의 미래 현금흐름을 현재가치로 할인

✔ 공식

DCF = Σ(현금흐름 / (1+할인율)^n)

✔ 해석

DCF > 현재가 → 저평가
DCF < 현재가 → 고평가

✔ 주의

할인율, 성장률 추정에 따라 크게 달라짐

## 4. 분석가 추천도 (Analyst Recommendation)
✔ 정의

투자은행/증권사 분석가의 매수/보유/매도 비율

✔ 해석

매수 비율 높음 → 긍정적 평가
매도 비율 높음 → 부정적 평가

✔ 주의

과거 분석가 정확도가 그다지 높지 않음

## 5. PEG Ratio (성장 고려 평가)
✔ 정의

성장률을 고려한 PER = PER / 성장률

✔ 해석

PEG < 1 → 성장 대비 저평가
PEG > 2 → 성장 대비 고평가

## 6. EV/EBITDA (기업가치 기준)
✔ 정의

산업별 표준화된 가치 비교

✔ 해석

동일 산업 평균보다 낮으면 저평가

# 🎯 평가 팩터 요약표

| 지표 | 의미 | 해석 |
|------|------|------|
| Target Price | 목표주가 | 현재가 < 목표가 = 저평가 |
| Relative Valuation | 상대평가 | 동종사 비교 |
| DCF / Absolute Valuation | 절대평가 | 미래현금흐름 기반 |
| Analyst Rating | 분석가 평가 | 전문가 의견 |
| PEG Ratio | 성장률 고려 평가 | 성장성 함께 고려 |
| EV/EBITDA | 표준화 평가 | 업종 간 비교 가능 |

# 🎯 최종 결론

평가 팩터는 **"현 주가의 공정성"**을 다각도로 검증하는 종합 평가 지표입니다.
VALUE 팩터가 "싼가"를 묻는다면, VALUATION 팩터는 "그게 정말 싼가"를 재확인하는 역할을 합니다.
여러 평가 방법을 종합해서 보면 더 정확한 투자 판단이 가능합니다.
