
# 💎 **VALUATION 팩터 (5개) — 초보자용 완전 정리**

Valuation 팩터는
**“이 기업이 지금 비싼가? 싼가?”**
즉 현재 주가가 기업의 가치를 얼마나 반영하고 있는지를 판단하는 지표들이다.

> 💡 핵심: 기업의 **내재가치 대비 주가 수준**을 평가한다.

---

# 🔵 **1) MAGIC_FORMULA (마법 공식)**

조엘 그린블라트의 전설적 가치투자 공식

### ✔ 정의

그린블라트가 제안한
**가치 + 수익성**을 동시에 반영하는 종합 순위 지표.

### ✔ 구성

1. **Earnings Yield(기업수익률)**
2. **ROIC(투하자본수익률)**

두 항목의 순위를 합산하여 점수가 낮을수록 우수.

### ✔ 공식

```
마법 공식 순위 = EY 순위 + ROIC 순위
```

### ✔ 해석

* 순위 낮음 → 매우 우량
* 순위 높음 → 비선호 기업

### ✔ 예시

* EY 순위 15위
* ROIC 순위 8위
  → 총점 23점 (상위 기업)

### ✔ 주의

* 금융/에너지 업종은 ROIC 해석 신중 필요
* 단독 사용보다 포트 전략에 적합

---

# 🔵 **2) GRAHAM_NUMBER (그레이엄 넘버)**

벤저민 그레이엄의 “적정 가치 계산법”

### ✔ 정의

그레이엄이 제시한 **내재가치의 최대 적정 가격**
→ “이 가격 이하이면 저평가일 수 있다”

### ✔ 공식

```
Graham Number = √(22.5 × EPS × BPS)
```

### ✔ 해석

* 주가 < 그레이엄 넘버 → 저평가 가능성
* 주가 > 그레이엄 넘버 → 고평가 가능성

### ✔ 예시

EPS 3,000원, BPS 20,000원
→ √(22.5 × 3000 × 20000)
≈ 36,742원

주가 30,000원 → 저평가

### ✔ 주의

* EPS 변동성이 큰 성장주는 적용 불리
* 기술주에는 잘 맞지 않음

---

# 🔵 **3) GREENBLATT_RANK (그린블랫 순위)**

### ✔ 정의

Magic Formula의 순위값 자체.
기업수익률(EY) + ROIC를 기반으로 기업을 **전체 순위화**.

### ✔ 공식

```
그린블랫 순위 = Magic Formula 점수를 전체 기업 중 순위화
```

### ✔ 해석

* 상위 10~20위 → 매우 우량
* 상위 50위 이내 → 좋은 기업
* 하위권 → 비선호 기업

### ✔ 예시

전체 2,000개 기업 중 12위 → 매우 우수

### ✔ 주의

* 순위는 시장 환경에 따라 변동 큼

---

# 🔵 **4) PS_RATIO (PSR: Price to Sales Ratio)**

### ✔ 정의

주가가 매출 대비 얼마나 비싼지 측정하는 지표.
→ 순이익이 적거나 없어도 평가 가능한 가치 지표.

### ✔ 공식

```
PSR = 시가총액 / 매출액
```

### ✔ 해석

* 1.0 이하 → 저평가
* 2.0~3.0 → 보통
* 5.0↑ → 성장주 or 고평가

### ✔ 예시

매출 5조, 시총 10조 → PSR=2.0
→ 적정 수준

### ✔ 주의

* 자산규모 작은 SaaS 기업은 PSR이 높아도 정상
* 순이익이 적을때 PER보다 PSR이 유리

---

# 🔵 **5) PRICE_TO_FCF (Price to FCF)**

### ✔ 정의

기업의 **잉여현금흐름(FCF)** 대비 주가가 얼마나 비싼지 판단.

### ✔ 공식

```
P/FCF = 시가총액 / FCF
```

### ✔ 해석

* 낮을수록 → 저평가
* 높을수록 → 미래 기대 반영

### ✔ 예시

FCF 1조, 시총 10조
→ P/FCF = 10

(보통 15 이하가 매력적인 구간)

### ✔ 주의

* 투자(설비) 많은 산업은 FCF가 일시적으로 왜곡됨
* 성장기 기업은 FCF가 음수일 수 있음

# 🎯 **VALUATION 팩터 요약**

| 팩터              | 핵심 의미            |
| --------------- | ---------------- |
| Magic Formula   | 가치+수익성 복합 순위     |
| Graham Number   | 적정 가치 최대 한계      |
| Greenblatt Rank | Magic Formula 순위 |
| PSR             | 매출 대비 주가 수준      |
| P/FCF           | 현금 대비 주가 수준      |

---

# 🎯 **최종 결론**

Valuation 팩터는 **현재 가격이 기업의 실질 가치와 비교해 비싼지 싼지**를 판단하는 핵심 도구다.

특히 실전에서 강력한 팩터는 다음 3개:

* **Price-to-FCF** → 가장 신뢰성 높은 가치 판단
* **PSR** → 이익 변동성이 큰 기업에서 유용
* **Magic Formula** → 가치 + 퀄리티 결합 전략의 핵심

이 팩터들은 VALUE, QUALITY 팩터와 조합했을 때
가장 강력한 퀀트 전략을 만든다.
