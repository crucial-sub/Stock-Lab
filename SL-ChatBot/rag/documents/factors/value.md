# 가치 투자 팩터 (Value Factors)

## 개요
가치 투자는 내재가치 대비 저평가된 주식을 발굴하는 전략입니다.

## 주요 팩터

### 1. PER (주가수익비율)
- **정의**: 주가를 주당순이익(EPS)으로 나눈 값
- **공식**: `주가 / EPS`
- **해석**: 낮을수록 저평가
- **기준값**:
  - 우량주: 5-15
  - 업종별로 차이 있음
- **주의사항**:
  - 적자 기업은 PER 계산 불가
  - 일시적 이익 급증 시 왜곡 가능

### 2. PBR (주가순자산비율)
- **정의**: 주가를 주당순자산(BPS)으로 나눈 값
- **공식**: `주가 / BPS`
- **해석**: 1 미만이면 청산가치 대비 저평가
- **기준값**:
  - 우량주: 0.5-1.5
  - 1 미만: 자산가치 대비 저평가
- **활용**: 자산주, 금융주 평가에 유용

### 3. PSR (주가매출비율)
- **정의**: 주가를 주당매출로 나눈 값
- **공식**: `주가 / 주당매출`
- **해석**: 낮을수록 저평가
- **기준값**: 0.5-2
- **활용**: 적자 기업이나 성장 초기 기업 평가

### 4. EV/EBITDA (기업가치배수)
- **정의**: 기업가치를 EBITDA로 나눈 값
- **공식**: `(시가총액 + 순차입금) / EBITDA`
- **해석**: 낮을수록 저평가
- **기준값**: 5-10
- **장점**:
  - 업종 간 비교 용이
  - 감가상각 영향 제거
  - 부채 수준 반영

## 전략 조합 예시

### 기본 가치주 전략
```json
{
  "buy_conditions": [
    {"factor": "PER", "operator": "<", "value": 15},
    {"factor": "PBR", "operator": "<", "value": 1.5},
    {"factor": "ROE", "operator": ">", "value": 10}
  ]
}
```

### 저PBR + 고ROE 전략
```json
{
  "buy_conditions": [
    {"factor": "PBR", "operator": "<", "value": 1.0},
    {"factor": "ROE", "operator": ">", "value": 15},
    {"factor": "부채비율", "operator": "<", "value": 100}
  ]
}
```

### 다중 밸류에이션 전략
```json
{
  "buy_conditions": [
    {"factor": "PER", "operator": "<", "value": 12},
    {"factor": "PBR", "operator": "<", "value": 1.2},
    {"factor": "PSR", "operator": "<", "value": 1.5},
    {"factor": "배당수익률", "operator": ">", "value": 3}
  ]
}
```

## 업종별 기준값

### 제조업
- PER: 8-12
- PBR: 0.8-1.5

### IT/테크
- PER: 15-25
- PBR: 2-4

### 금융
- PER: 5-10
- PBR: 0.5-1.0

### 유틸리티
- PER: 10-15
- PBR: 1-2

## 주의사항

1. **회계 조작 가능성**: 일회성 이익으로 지표 왜곡 가능
2. **업황 사이클**: 업종 사이클 고려 필요
3. **가치 함정**: 저평가지만 펀더멘털 악화 주의
4. **시장 환경**: 금리 상승기에 유리, 성장장에서는 부진
