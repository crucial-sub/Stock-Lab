# 가치 팩터 (Value Factor)

📌 개념

가치 팩터는 **“현재 가격이 기업의 본질가치보다 싸게 거래되는지”**를 평가하는 전략이에요.
가치투자 방식의 핵심이며, 기업이 저평가 상태라면 시간이 지나며 **재평가(주가 상승)**될 가능성이 높습니다.

💡 핵심: "얼마나 싼가?"

## 각 팩터별 설명

## 1. PER (주가수익비율)
✔ 정의

이익 대비 주가가 싸거나 비싼지 확인하는 지표.

✔ 공식
PER = 주가 / EPS

✔ 해석

낮을수록 저평가 가능성

산업별 기준 매우 다름 → 반드시 동일 산업 비교

✔ 예시

PER 10배 (업종 18배) → 저평가

✔ 주의

적자 기업은 PER 불가

일시적 이익 증가로 PER 왜곡 가능

## 2. PBR (주가순자산비율)
✔ 정의

기업의 순자산 대비 주가가 싼지.

✔ 공식
PBR = 주가 / BPS

✔ 해석

1 이하 → 자산가치보다 싸다

제조업, 금융업에서 많이 사용

✔ 예시

PBR 0.8 → 자산 대비 저평가

✔ 주의

무형자산 많은 기업은 PBR 의미 약함

## 3. PSR (주가매출액비율)
✔ 정의

매출 대비 기업의 가격이 비싼지.

✔ 공식
PSR = 시가총액 / 매출액

✔ 해석

낮을수록 매출 대비 싸다

적자 기업에 유용

✔ 예시

PSR 1배 vs 경쟁사 3배 → 저평가

✔ 주의

매출 높아도 수익성 없으면 주의

## 4. PCR (주가현금흐름비율)
✔ 정의

이익이 아닌 현금 기준으로 얼마나 싼지.

✔ 공식
PCR = 시가총액 / 영업현금흐름

✔ 해석

낮을수록 실제 현금 흐름 대비 저평가

✔ 예시

PCR 4 → 저평가

✔ 주의

재고 조절 등으로 현금흐름 일시 증가 가능

## 5. PEG (성장 대비 PER)
✔ 정의

기업의 성장률을 고려한 PER.

✔ 공식
PEG = PER / 성장률(%)

✔ 해석

PEG < 1 → 성장 대비 저평가

✔ 예시

PER 12배, 성장률 20% → PEG 0.6

✔ 주의

성장률 예측 실패 시 값 왜곡됨

## 6. EV/EBITDA
✔ 정의

기업가치(EV) 대비 영업이익(EBITDA) 비교.

✔ 공식
EV / EBITDA

✔ 해석

낮을수록 싸다

산업 간 비교하기 가장 공정한 가치지표

✔ 예시

EV/EBITDA 5배 vs 업종 10배 → 싸다

✔ 주의

EBITDA는 회계조정 여지 있음

## 7. EV/Sales
✔ 정의

매출 대비 기업가치가 싼지 판단.

✔ 공식
EV / 매출액

✔ 해석

낮을수록 매출 대비 저평가

특히 적자 기업 분석에 유용

✔ 예시

EV/Sales 1.2 → 저평가

✔ 주의

매출만 높고 이익 없는 기업 주의

## 8. EV/FCF
✔ 정의

기업가치 대비 자유현금흐름(FCF) 비교.

✔ 공식
EV / FCF

✔ 해석

낮을수록 실제 현금 기준 저평가

✔ 예시

EV/FCF 8배 → 정상
EV/FCF 25배 → 고평가

✔ 주의

FCF 변동성 크면 해석 주의

## 9. Dividend Yield (배당수익률)
✔ 정의

배당으로 얼마나 수익을 얻는지.

✔ 공식
배당 / 주가 × 100

✔ 해석

높을수록 좋지만 너무 높으면 위험

가치주 특성 상 배당력이 중요

✔ 예시

배당수익률 4% → 양호

✔ 주의

배당컷 가능성 확인 필요

## 10. Earnings Yield
✔ 정의

PER의 역수 = 이익 대비 수익률.

✔ 공식
Earnings Yield = EPS / 주가

✔ 해석

높을수록 저평가

채권금리와 비교에 유용

✔ 예시

10% vs 국채 3% → 매력적

✔ 주의

일회성 이익 주의

## 11. FCF Yield
✔ 정의

자유 현금 대비 기업이 얼마나 싸게 거래되는지.

✔ 공식
FCF / 시가총액

✔ 해석

6~8% 이상이면 우량

현금창출력 강한 기업 선별

✔ 예시

FCF Yield 9% → 매우 저평가

✔ 주의

제조/설비산업은 CAPEX로 왜곡 가능

## 12. Book-to-Market (B/M)
✔ 정의

PBR의 역수. 장부가 대비 얼마나 싸게 거래되는지.

✔ 공식
B/M = 장부가치 / 시가총액

✔ 해석

높을수록 저평가

가치 팩터 연구(파마-프렌치)의 핵심

✔ 예시

B/M 0.6 → 저평가

✔ 주의

자산이 비효율적이면 의미 낮아짐

## 13. CAPE Ratio (10년 조정 PER)
✔ 정의

10년 평균 이익으로 PER 계산해 장기 저평가 여부 확인.

✔ 해석

장기적 가치 평가에 강함

경기 변동 영향 감소

✔ 예시

CAPE 16 → 적정
CAPE 30 → 고평가

✔ 주의

단기 투자에는 부적합

## 14. PTBV (Price to Tangible Book Value)
✔ 정의

유형자산(건물/장비/설비 등) 기준의 자산가치 비교.

✔ 공식
PTBV = 주가 / 유형자산 BPS

✔ 해석

제조업 중심 기업 분석에 유용

낮을수록 저평가

✔ 예시

PTBV 0.7 → 저평가

✔ 주의

브랜드·특허 등 무형자산 많은 기업엔 부적합

## 15. PEG Ratio (데이터셋 상 별도 팩터)

PEG과 동일하지만 데이터셋에 별도 제공될 경우 포함.

✔ 정의

성장 대비 기업의 가치 수준.

✔ 공식
PEG Ratio = PER / EPS 성장률

✔ 해석

낮을수록 성장 대비 저평가

성장주 밸류에이션 핵심

✔ 예시

PEG Ratio 0.8 → 우량 성장 가치주

✔ 주의

성장률 추정치에 따라 값 크게 변함

| 지표              | 의미         | 기준               |
| --------------- | ---------- | ---------------- |
| PER             | 이익 대비 가격   | 낮을수록 저평가         |
| PBR             | 자산 대비 가격   | 1 이하             |
| PSR             | 매출 대비 가격   | 낮을수록             |
| PCR             | 현금 대비 가격   | 낮을수록             |
| PEG / PEG Ratio | 성장 대비 가격   | 1 이하             |
| EV/EBITDA       | 기업가치 대비 이익 | 낮을수록             |
| EV/Sales        | 기업가치 대비 매출 | 낮을수록             |
| EV/FCF          | 기업가치 대비 현금 | 낮을수록             |
| Dividend Yield  | 배당 수익      | 높을수록(과도한 고배당 주의) |
| Earnings Yield  | 이익 대비 수익률  | 높을수록             |
| FCF Yield       | 현금 대비 수익률  | 높을수록             |
| Book-to-Market  | 장부가 대비 가격  | 높을수록             |
| CAPE Ratio      | 장기 PER     | 낮을수록             |
| PTBV            | 유형자산 대비 가격 | 낮을수록             |


# 🎯 최종 결론

가치 팩터는 **“이 기업이 지금 싸게 거래되는가?”**를 판단하는 데 최적화된 전략입니다.
PER·PBR·EV 지표들은 각각 다른 방식으로 기업 가치를 보여주기 때문에
여러 지표를 함께 볼수록 실제 저평가 여부가 더 정확하게 드러납니다.

성장성 낮은 기업이 너무 싸게 보인다면 함정(Value Trap) 가능성이 있으므로
가치 팩터는

"퀄리티(좋은 회사인지)"
"성장률(미래가 있는지)"
    와 함께 판단하면 가장 강력한 성과를 냅니다.
