# 🌪 **VOLATILITY 팩터 (6개) **

변동성 팩터는 “이 종목이 얼마나 흔들리고 있는가?”,
즉 **가격의 움직임 크기와 위험 수준**을 측정하는 지표다.

> 💡 핵심: 변동성이 크면 위험 + 기회,
> 변동성이 낮으면 안정 + 방향성 약함.

---

# 🔵 **1) VOLATILITY_20D (20일 변동성)**

### ✔ 정의

최근 20일 동안의 **일일 수익률 표준편차**.
→ 한 달 동안 가격이 얼마나 출렁였는지.

### ✔ 공식

```
20일 변동성 = 20일 수익률의 표준편차
```

### ✔ 해석

* 높을수록 가격 출렁임 큼 → 위험 ↑
* 낮을수록 안정적 → 박스권 가능성

### ✔ 예시

20일 변동성 3% → 안정
12% → 고위험

### ✔ 주의

* 단기간 급등/급락이 있으면 왜곡 가능

---

# 🔵 **2) VOLATILITY_60D (60일 변동성)**

### ✔ 정의

최근 60일(약 3개월) 동안의 가격 변동성.

### ✔ 의미

* 20일보다 “중기 리스크”를 잘 반영
* 변동성 추세를 확인할 때 사용

### ✔ 해석

* 상승장 초반: 변동성 낮음
* 급등·급락 구간: 변동성 폭발

### ✔ 주의

* 변동성이 높다고 무조건 나쁜 건 아님
  (신규 트렌드 시작일 수도)

---

# 🔵 **3) HISTORICAL_VOLATILITY_20 (20일 역사적 변동성)**

### ✔ 정의

20일 변동성에 **연환산(×√252)** 을 적용한 지표.
→ “연 기준으로 환산하면 어느 정도 위험인가?”

### ✔ 공식

```
HV20 = 표준편차(20일 수익률) × √252
```

### ✔ 해석

* 20% 이하 → 안정
* 40% 이상 → 고변동 종목

### ✔ 예시

단기 변동성 2.5%이면
연 변동성은 약 40% 수준.

### ✔ 주의

* 단기 급등주(테마주)는 HV가 매우 큼

---

# 🔵 **4) HISTORICAL_VOLATILITY_60 (60일 역사적 변동성)**

### ✔ 정의

60일 변동성을 연환산한 값.

### ✔ 의미

* 중기 트렌드의 위험도 판단
* 옵션 가격 산정 때도 쓰이는 표준적인 지표

### ✔ 해석

* 30% 이하 → 안정
* 50%↑ → 변동 큰 종목

---

# 🔵 **5) PARKINSON_VOLATILITY (고가–저가 기반 변동성)**

### ✔ 정의

수익률 대신 **고가/저가 차이**만으로 계산한 변동성.
→ 일중 변동폭이 큰 종목을 잘 포착.

### ✔ 공식

```
Parkinson = √( Σ(ln(고가/저가))² / (4 ln2) )
```

### ✔ 해석

* 장중 변동성이 큰 종목 → 값 크게 증가
* 갭상승/갭하락 역시 반영

### ✔ 예시

장중 고저 차이가 10% 이상 자주 발생 → Parkinson 변동성 ↑

### ✔ 주의

* 종가 기반 변동성과 값이 다를 수 있음
  (장중만 많이 흔들리는 종목에서 특히 차이 큼)

---

# 🔵 **6) WEEK_52_POSITION (52주 High/Low 대비 위치)**

### ✔ 정의

지난 1년(52주) 동안의 최저~최고 범위 안에서
현재 주가가 어디에 있는지.

### ✔ 공식

```
(현재가 - 52주 최저가) / (52주 최고가 - 52주 최저가)
```

### ✔ 해석

* 0.8↑ → 52주 최고가 근처(강한 종목)
* 0.2↓ → 52주 최저가 근처(약한 종목)

### ✔ 예시

52주 고가 10만원
52주 저가 5만원
현재가격 9만원

→ 위치 = 0.8 (상단, 강세)

### ✔ 주의

* 신저가 근처 종목은 “싸다”가 아니라 **위험**일 수도 있음
* 신고가 근처 종목은 **추세 강한 강세주** 특징

# 🎯 **VOLATILITY 팩터 요약표**

| 팩터            | 의미           | 핵심 포인트     |
| ------------- | ------------ | ---------- |
| 20일 변동성       | 단기 변동 크기     | 한 달간 위험 수준 |
| 60일 변동성       | 중기 변동 크기     | 변동성 추세     |
| 역사적 변동성 20/60 | 연간으로 환산한 변동성 | 중·장기 리스크   |
| Parkinson     | 고저차 기반 변동성   | 일중 흔들림 강조  |
| 52주 위치        | 1년 중 현재 위치   | 강/약세 구간 판단 |

---

# 🎯 **최종 결론**

변동성 팩터는 **리스크 + 기회**를 동시에 보여준다.
특히 아래 3개는 실전에서 매우 자주 사용된다:

* **52주 High/Low 대비 위치 (WEEK_52_POSITION)**
* **20일 변동성 / 60일 변동성**
* **Parkinson 변동성**

변동성이 높으면 위험하지만 단기 급등 기회도 많고,
변동성이 낮으면 안전하지만 수익 기회는 제한적이다.

👉 결국 변동성 팩터는 **투자 전략의 위험 관리 핵심 도구**다.
