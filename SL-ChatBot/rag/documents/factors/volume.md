# 📦 **VOLUME 팩터 (7개) **

거래량 팩터는 **“얼마나 활발하게 거래되고 있는가?”, “수급이 강한가?”** 를 판단하는 지표들이야.

> 💡 핵심: 가격은 속일 수 있어도 **거래량은 속이기 어렵다.**
> 거래량 기반 지표는 “진짜 수급”을 볼 수 있어서 트레이더들이 특히 좋아함.

---

# 🔵 **1) AVG_TRADING_VALUE (평균 거래대금)**

### ✔ 정의

최근 일정 기간 동안 하루 평균 거래된 금액(원 단위).
→ “이 종목은 얼마나 활발하게 거래되는가?”

### ✔ 공식

```
평균 거래대금 = 최근 N일 거래대금 평균
```

### ✔ 해석

* 높을수록 유동성 좋음
* 낮으면 매수·매도 체결이 어려울 수 있음

### ✔ 예시

평균 거래대금 500억 → 매우 활발
평균 거래대금 5억 → 비유동성, 위험

### ✔ 주의

* 하루만 거래량 급등해도 평균이 왜곡될 수 있음

---

# 🔵 **2) TURNOVER_RATE_20D (20일 회전율)**

### ✔ 정의

전체 상장주식 수 대비 최근 20일 동안 얼마나 거래되었는지를 보는 지표.
→ “주주들이 얼마나 활발히 주식을 사고팔았는가?”

### ✔ 공식

```
20일 회전율 = (최근 20일 평균 거래량 / 상장주식수) × 100
```

### ✔ 해석

* 높으면 → 단기 관심 집중(테마, 모멘텀)
* 너무 높으면 → 단기 급등락 위험

### ✔ 예시

20일 회전율 30% → 매우 활발
5% → 평범함

### ✔ 주의

상장주식수가 큰 기업(삼성전자)은 회전율이 낮게 나오는 구조.

---

# 🔵 **3) VOLUME_RATIO_20D (20일 거래량 비율)**

### ✔ 정의

현재 거래량이 20일 평균 대비 얼마나 많은지를 측정.
→ “오늘 거래량이 평소보다 많은가?”

### ✔ 공식

```
거래량 비율 = 현재 거래량 / 20일 평균 거래량
```

### ✔ 해석

* 2.0 → 오늘 거래량이 보통의 2배
* 5.0 → 강한 관심(급등 전형)
* 0.5 → 관심 부족

### ✔ 예시

평균 10만주인데 오늘 100만주 → 10배 증가

### ✔ 주의

악재 뉴스로 거래량 터질 때도 있음(무조건 상승 아님).

# 🔵 **5) VOLUME_MA_20 (거래량 20일 평균)**

(5일/20일은 기간만 다르고 구조는 동일)

### ✔ 정의

최근 N일 동안의 평균 거래량.
→ 단기/중기 수급 추세 확인.

### ✔ 공식

```
VOLUME_MA_N = 최근 N일 거래량 평균
```

### ✔ 해석

* 거래량이 5일선·20일선 위로 증가 → 수급 강화
* 아래로 감소 → 관심 감소

### ✔ 예시

5일 이동평균 거래량보다 오늘 거래량이 3배 ↑ → 강한 모멘텀

### ✔ 주의

장기간 거래량 없는 종목은 해석 무의미.

---

# 🔵 **6) VOLUME_SURGE (거래량 급증률)**

### ✔ 정의

특정일에 거래량이 얼마나 폭발적으로 증가했는지를 측정.
→ 급등 전 “선행 신호”로 자주 등장.

### ✔ 공식

```
거래량 급증률 = 오늘 거래량 / 20일 평균 거래량
```

### ✔ 해석

* 3배↑ → 강한 관심 / 뉴스 발생 가능
* 5배↑ → 테마주 가능성
* 10배↑ → 위험 + 기회(투기성)

### ✔ 예시

평균 10만주 → 오늘 100만주 → 10배 급증

### ✔ 주의

거래량 급증 = “반드시 상승” 이 아님
→ **악재로 인한 공포 매도**일 수도 있음.

---

# 🔵 **7) ACCUMULATION_DISTRIBUTION (A/D Line)**

### ✔ 정의

매수세/매도세를 **가격 + 거래량**으로 판단하는 지표.
→ 가격의 “진짜 방향성”을 보여줌.

### ✔ 공식 (간단 버전)

```
AD = 누적{ ((종가 - 저가) - (고가 - 종가)) / (고가 - 저가) × 거래량 }
```

### ✔ 해석

* A/D 상승 → 매집(조용히 사는 세력)
* A/D 하락 → 분산(조용히 파는 세력)

### ✔ 예시

가격은 횡보인데 A/D가 급상승 → “수급 매집 신호”

### ✔ 주의

고가=저가 같은 비정상 데이터에서는 왜곡 가능.

# 🎯 **VOLUME 팩터 요약**

| 팩터      | 핵심 의미       |
| ------- | ----------- |
| 평균 거래대금 | 유동성, 거래 활발도 |
| 회전율     | 투자자 교체 속도   |
| 거래량 비율  | 오늘 관심도      |
| 거래량 MA  | 단기/중기 수급 추세 |
| 거래량 급증률 | 뉴스·테마·급변 신호 |
| A/D 라인  | 숨겨진 매집/분산   |

---

# 🎯 **최종 결론**

거래량 팩터는 주가 움직임의 **진짜 힘(수급)** 을 보여준다.
특히 다음 4개가 실전에서 가장 중요하다:

* **VOLUME_SURGE (급증률)**
* **ACCUMULATION_DISTRIBUTION (A/D Line)**
* **VOLUME_RATIO_20D (평균 대비 거래량 비율)**
* **TURNOVER_RATE_20D (20일 회전율)**

이 지표들은 “진짜 매수세가 존재하는가?” 를 판단하는 데 필수다.
