# 투자 조언 금지 정책 🚫

## 개념

Stock Lab 챗봇은 **교육 목적의 정보 제공 서비스**입니다.
법적 규제에 따라 **특정 종목 추천이나 투자 조언은 절대 제공하지 않습니다**.

> ⚠️ **핵심 원칙**: 정보 제공 ✅ | 투자 조언 ❌

---

## 법적 배경

### 자본시장법 규정
- **무인가 투자자문업 금지**: 금융위원회 인가 없이 투자 조언 불가
- **처벌 규정**: 5년 이하 징역 또는 2억원 이하 벌금
- **적용 범위**: 유료/무료 관계없이 모든 투자 조언 행위

### 금융소비자보호법
- **설명의무**: 금융상품의 위험성 충분히 설명
- **적합성 원칙**: 개인 투자자 상황 고려 필수
- **숙려 기간**: 투자 결정 전 충분한 검토 시간 보장

## 금지 행위 명세 ❌

### 1. 종목 추천 금지
```
❌ 금지된 표현:
- "삼성전자를 사세요"
- "이 종목은 상승할 것입니다"
- "지금이 매수 타이밍입니다"
- "이 주식은 저평가되어 있습니다"
- "포트폴리오에 꼭 포함시키세요"

✅ 허용된 표현:
- "PER이 낮은 종목을 찾는 방법은..."
- "일반적으로 가치투자 전략은..."
- "재무제표 분석 방법은..."
```

### 2. 매매 시점 조언 금지
```
❌ 금지된 표현:
- "지금 매수/매도하세요"
- "○○원이 적정 매수가입니다"
- "△△% 수익에서 매도하세요"
- "손절선은 ××원으로 설정하세요"

✅ 허용된 표현:
- "일반적인 손절 원칙은..."
- "리스크 관리 방법으로는..."
- "투자 타이밍 판단 기준은..."
```

### 3. 수익률 보장 금지
```
❌ 금지된 표현:
- "연 20% 수익 보장"
- "손실 위험이 없습니다"
- "확실한 수익을 얻을 수 있습니다"
- "이 전략은 항상 수익이 납니다"

✅ 허용된 표현:
- "과거 백테스트 결과는... (과거 성과는 미래를 보장하지 않음)"
- "이론적 기대 수익률은..."
- "리스크와 수익의 관계는..."
```

### 4. 개인화된 조언 금지
```
❌ 금지된 표현:
- "당신의 경우 이 전략이 적합합니다"
- "귀하의 나이에는 공격적 투자를..."
- "월 100만원이면 이렇게 투자하세요"

✅ 허용된 표현:
- "일반적으로 연령대별 자산배분은..."
- "투자 성향 테스트 방법은..."
- "포트폴리오 이론에 따르면..."
```

## 허용된 서비스 범위 ✅

### 1. 교육 콘텐츠 제공
- **개념 설명**: 금융 용어, 투자 이론
- **방법론 교육**: 분석 방법, 전략 유형
- **도구 사용법**: 차트 읽기, 지표 계산

### 2. 일반 정보 제공
- **시장 데이터**: 지수, 환율, 금리
- **공시 정보**: 재무제표, 공시 내용
- **통계 자료**: 업종 평균, 시장 통계

### 3. 분석 도구 제공
- **스크리닝 기능**: 조건 검색 도구
- **백테스트**: 과거 데이터 검증
- **리스크 측정**: 변동성, 상관관계

### 4. 중립적 비교
- **전략 비교**: 장단점 객관적 제시
- **상품 비교**: 특징 나열 (추천 없이)
- **방법론 비교**: 각 방법의 특성

## 응답 템플릿

### 종목 문의 시
```markdown
죄송합니다. 저는 특정 종목에 대한 투자 조언을 제공할 수 없습니다.

대신 도움드릴 수 있는 것:
✅ 종목 분석 방법 설명
✅ 재무제표 읽는 법
✅ 투자 지표 계산 방법
✅ 리스크 관리 원칙

투자 결정은 반드시 본인의 판단으로 하시기 바랍니다.
```

### 매매 타이밍 문의 시
```markdown
매매 시점은 개인이 직접 판단하셔야 합니다.

참고할 수 있는 일반 원칙:
📊 기술적 분석 지표 활용법
📈 펀더멘털 분석 기준
⚖️ 리스크/수익 비율 고려
🎯 개인 투자 목표 설정

구체적인 투자 조언은 인가받은 투자자문사에 문의하세요.
```

### 포트폴리오 구성 문의 시
```markdown
포트폴리오 구성은 개인 상황에 따라 다릅니다.

일반적인 포트폴리오 이론:
📚 현대 포트폴리오 이론 (MPT)
🔄 자산 배분 전략
📊 분산 투자 원칙
⚖️ 리밸런싱 방법

개인 맞춤 조언은 전문 자문사와 상담하세요.
```

## 필터링 시스템

### 1단계: 키워드 감지
```python
금지_키워드 = [
    "추천", "사세요", "파세요",
    "매수하세요", "매도하세요",
    "투자하세요", "보장", "확실"
]
```

### 2단계: 문맥 분석
- 특정 종목 언급 + 행동 유도
- 시점 특정 + 매매 제안
- 수익률 + 보장/확실

### 3단계: 응답 차단
- 금지 패턴 감지 시 표준 거부 메시지
- 교육 콘텐츠로 대체 응답

## 면책 조항 (모든 응답 포함)

```
📌 중요 안내
- 본 정보는 교육 목적으로만 제공됩니다
- 투자 조언이나 추천이 아닙니다
- 투자 손실 책임은 투자자 본인에게 있습니다
- 투자 전 충분한 검토와 전문가 상담을 권합니다
```

## 예외 상황

### 허용되는 경우
1. **학술 연구**: 논문, 연구 결과 인용
2. **역사적 사례**: 과거 사례 교육적 설명
3. **가상 시뮬레이션**: 모의투자, 교육용 예시

### 주의 필요 상황
1. **뉴스 인용**: 팩트만 전달, 의견 배제
2. **전문가 의견**: 출처 명시, 조언 아님 명시
3. **통계 제시**: 해석 없이 데이터만 제공

## 위반 시 조치

### 시스템 차원
1. **자동 차단**: 금지 패턴 감지 시 응답 차단
2. **로그 기록**: 위반 시도 모니터링
3. **패턴 학습**: 새로운 우회 시도 감지

### 사용자 보호
1. **경고 메시지**: 위험성 안내
2. **대안 제시**: 올바른 정보 획득 방법
3. **전문가 안내**: 공인 자문사 연결
