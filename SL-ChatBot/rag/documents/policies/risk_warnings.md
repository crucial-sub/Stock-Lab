# 투자 위험 안내 ⚠️

## 개념

모든 투자에는 **원금 손실 위험**이 있습니다.
Stock Lab은 사용자가 **충분한 위험 인지** 하에 투자 결정을 내릴 수 있도록 명확한 위험 안내를 제공합니다.

> 🚨 **핵심 메시지**: 투자의 모든 책임은 투자자 본인에게 있습니다

---

## 주요 투자 위험

### 1. 시장 위험 (Market Risk) 📉

#### 정의
전체 시장 하락으로 인한 손실 위험

#### 특징
- **체계적 위험**: 분산투자로도 완전 제거 불가
- **경제 영향**: 경기 침체, 금리 인상, 인플레이션
- **글로벌 연동**: 해외 시장 영향 직접 전달

#### 대응 방법
```
✅ 장기 투자 관점 유지
✅ 적절한 현금 비중 유지
✅ 정기적 리밸런싱
⚠️ 단기 변동에 과민 반응 금지
```

### 2. 개별 종목 위험 (Specific Risk) 🎯

#### 정의
특정 기업의 고유 위험

#### 주요 유형
| 위험 유형 | 설명 | 예시 |
|----------|------|------|
| **경영 위험** | 잘못된 경영 판단 | 사업 실패, M&A 실패 |
| **재무 위험** | 재무구조 악화 | 부채 증가, 유동성 위기 |
| **산업 위험** | 업종 전체 침체 | 규제 강화, 기술 변화 |
| **규제 위험** | 정부 정책 변화 | 세금 인상, 영업 제한 |

### 3. 유동성 위험 (Liquidity Risk) 💧

#### 정의
원하는 시점에 적정 가격으로 매매할 수 없는 위험

#### 발생 상황
- **거래량 부족**: 소형주, 비우량주
- **시장 충격**: 급락장, 서킷브레이커
- **상하한가**: 연속 하한가로 매도 불가

#### 주의사항
```
⚠️ 급전이 필요한 자금 투자 금지
⚠️ 거래량 적은 종목 주의
⚠️ 시장가 주문 시 슬리피지 발생
```

### 4. 환율 위험 (Currency Risk) 💱

#### 적용 대상
- 해외 주식 직접 투자
- 해외 ETF 투자
- 외화 자산 보유

#### 영향 요인
```
환율 상승 → 평가 이익
환율 하락 → 평가 손실
(주가 상승해도 환율 하락으로 손실 가능)
```

### 5. 금리 위험 (Interest Rate Risk) 📊

#### 영향 관계
- **금리 상승**: 주식 가치 하락 압력
- **금리 하락**: 주식 가치 상승 요인

#### 민감 업종
- 금융주 (은행, 보험)
- 부채 많은 기업
- 성장주 (미래 이익 할인)

## 위험 등급별 설명

### 🟢 초저위험 (1등급)
- **상품**: 예금, 적금
- **손실 가능성**: 거의 없음 (예금자보호)
- **적합 투자자**: 원금 보전 최우선

### 🟡 저위험 (2등급)
- **상품**: 국공채, 우량 회사채
- **손실 가능성**: 매우 낮음
- **적합 투자자**: 안정 추구형

### 🟠 중위험 (3등급)
- **상품**: 대형 우량주, 배당주
- **손실 가능성**: 보통 (-20% 내외)
- **적합 투자자**: 위험 중립형

### 🔴 고위험 (4등급)
- **상품**: 중소형주, 성장주
- **손실 가능성**: 높음 (-30% 이상)
- **적합 투자자**: 적극 투자형

### ⚫ 초고위험 (5등급)
- **상품**: 신생 기업, 테마주, 파생상품
- **손실 가능성**: 매우 높음 (-50% 이상)
- **적합 투자자**: 공격 투자형

## 손실 시나리오별 경고

### 시나리오 1: 단기 급락 ⚡
```
상황: 하루 -10% 하락
영향: 1000만원 → 900만원
심리: 공포, 패닉 매도 충동
대응:
✅ 투자 이유 재점검
✅ 기업 펀더멘털 확인
❌ 감정적 매도 금지
```

### 시나리오 2: 장기 침체 📉
```
상황: 6개월간 -30% 하락
영향: 1000만원 → 700만원
심리: 좌절, 투자 포기 고려
대응:
✅ 추가 매수 기회 검토
✅ 포트폴리오 리밸런싱
❌ 전량 손절 금지
```

### 시나리오 3: 개별 종목 폭락 💥
```
상황: 특정 종목 -50% 이상
영향: 심각한 포트폴리오 훼손
원인: 실적 쇼크, 스캔들, 상장폐지
대응:
✅ 냉정한 원인 분석
✅ 회복 가능성 평가
⚠️ 분산투자 중요성 재인식
```

```markdown
⚠️ 투자 위험 안내
• 모든 투자는 원금 손실 위험이 있습니다
• 과거 수익률이 미래 수익을 보장하지 않습니다
• 투자 결정과 그 결과는 투자자 본인의 책임입니다

```

### 고위험 상품 언급 시
```markdown
🚨 고위험 투자 경고

• 투자금 전액을 잃을 수 있습니다
• 충분한 지식과 경험이 필요합니다
• 반드시 소액으로 시작하시기 바랍니다
```

### 레버리지/파생상품 언급 시
```markdown
⛔ 초고위험 경고
• 레버리지는 손실을 확대시킵니다
• 원금 이상의 손실이 발생할 수 있습니다

```

## 연령대별 위험 안내

### 20-30대
```
💡 주요 위험 요인
• 과도한 리스크 추구 경향
• 단기 수익 집착
• 경험 부족으로 인한 실수

⚠️ 특별 주의사항
• 빚투자(신용/대출) 절대 금지
• 여유자금의 30% 이하로 시작
• 실패도 수업료라는 마음가짐
```

### 40-50대
```
💡 주요 위험 요인
• 은퇴 자금 손실 위험
• 자녀 교육비 압박
• 회복 시간 부족

⚠️ 특별 주의사항
• 안정성과 수익성 균형
• 은퇴 자금 별도 관리
• 급격한 전략 변경 자제
```

### 60대 이상
```
💡 주요 위험 요인
• 노후 자금 고갈 위험
• 건강 문제로 인한 급전 필요
• 투자 기간 제한

⚠️ 특별 주의사항
• 원금 보전 최우선
• 배당/이자 수익 중심
• 복잡한 상품 투자 자제
```

## 심리적 위험 관리

### 주요 투자 심리 함정

#### 1. 손실 회피 편향
- **증상**: 손실을 과도하게 두려워함
- **문제**: 좋은 기회 놓침
- **해결**: 계획된 리스크 수용

#### 2. 확증 편향
- **증상**: 유리한 정보만 선택적 수용
- **문제**: 객관적 판단 불가
- **해결**: 반대 의견도 검토

#### 3. 과신 편향
- **증상**: 자신의 능력 과대평가
- **문제**: 과도한 리스크 감수
- **해결**: 겸손한 자세 유지

#### 4. 군중 심리
- **증상**: 다수를 무조건 따라감
- **문제**: 고점 매수, 저점 매도
- **해결**: 독립적 사고 유지

## 긴급 상황 대응 가이드

### 시장 폭락 시 (-20% 이상)
1. **즉시 확인**: 폭락 원인 파악
2. **포트폴리오 점검**: 보유 종목 영향도
3. **현금 확인**: 추가 매수 여력
4. **의사결정**:
   - 우량주 추가 매수
   - 현 상태 유지
   - 부분 손절 (최후 수단)

### 개별 종목 악재 시
1. **사실 확인**: 루머 vs 사실
2. **영향 평가**: 일시적 vs 구조적
3. **대응 결정**:
   - 일시적: 유지 또는 추가 매수
   - 구조적: 즉시 손절

## 위험 관리 도구

### 1. 손절선 설정
```
기본 원칙: -7% ~ -10%
공식: 손절가 = 매수가 × (1 - 손절률)
예시: 10,000원 × 0.93 = 9,300원
```

### 2. 분산 투자
```
종목 수: 최소 10개 이상
종목당 비중: 최대 10% 이하
업종 분산: 3개 이상 업종
```

### 3. 리밸런싱
```
주기: 분기별 또는 반기별
기준: 목표 비중 ±5% 이탈 시
방법: 비중 초과 매도, 부족 매수
```

## 법적 책임 한계

### Stock Lab 면책사항
1. **정보 제공 한계**: 교육 목적 정보만 제공
2. **투자 조언 부재**: 매매 추천 불가
3. **손실 책임**: 모든 투자 손실은 투자자 책임
4. **정확성 한계**: 정보 오류 가능성 존재

### 투자자 책임
1. **자기 책임 원칙**: 모든 투자 결정은 본인 책임
2. **정보 검증 의무**: 제공 정보 교차 확인 필요
3. **적합성 판단**: 본인 상황에 맞는지 스스로 판단
4. **전문가 상담**: 필요시 공인 투자자문 이용
