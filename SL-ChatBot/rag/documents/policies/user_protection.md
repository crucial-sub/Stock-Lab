# 사용자 보호 정책 🛡️

## 개념

Stock Lab은 **사용자의 안전과 권익 보호**를 최우선으로 합니다.
금융 정보 서비스로서 사용자의 재산, 개인정보, 투자 안전을 종합적으로 보호합니다.

> 🔐 **핵심 가치**: 안전 | 투명성 | 책임 | 신뢰

---

## 1. 개인정보 보호 🔒

### 수집 원칙
- **최소 수집**: 서비스 제공에 필수적인 정보만 수집
- **목적 명시**: 수집 목적을 명확히 고지
- **동의 기반**: 사용자 동의 없이 수집 금지

### 수집하지 않는 정보
```
❌ 절대 수집 금지:
- 주민등록번호
- 계좌번호
- 비밀번호
- 신용카드 정보
- 휴대폰 번호
- 주소
- 가족 정보
- 소득 정보
- 자산 규모
```

### 수집 가능 정보 (동의 시)
```
✅ 최소 정보:
- 닉네임 (익명)
- 이메일 (선택)
- 투자 경험 수준 (초급/중급/고급)
- 관심 분야 (선택)
```

### 정보 보호 조치
| 조치 | 내용 |
|------|------|
| **암호화** | 모든 개인정보 암호화 저장 |
| **접근 제한** | 권한 있는 담당자만 접근 |
| **보관 기간** | 목적 달성 후 즉시 파기 |
| **제3자 제공** | 원칙적 금지 (법령 예외) |

## 2. 금융 피해 예방 💰

### 사기 예방 시스템

#### 1단계: 사전 차단
```python
사기_패턴_감지 = [
    "계좌 이체 요구",
    "투자금 입금 요청",
    "수수료 요구",
    "회원 가입비",
    "프리미엄 서비스 결제"
]
→ 자동 차단 및 신고
```

#### 2단계: 의심 패턴 경고
- 비정상적 수익률 약속
- 긴급성 조장
- 개인 연락 유도
- 외부 사이트 유도

#### 3단계: 사용자 교육
- 투자 사기 유형 안내
- 피해 예방 가이드
- 신고 방법 안내

### 리스크 등급 시스템

#### 사용자 평가
```
초급 투자자 → 고위험 상품 경고 강화
중급 투자자 → 표준 위험 안내
고급 투자자 → 상세 리스크 정보 제공
```

#### 자동 보호 기능
- **초급자 보호**: 파생상품, 레버리지 정보 차단
- **과도한 집중 경고**: 단일 종목 과다 관심 시 경고
- **손실 한도 알림**: 리스크 관리 교육 제공

## 4. 취약 계층 보호 👥

### 고령 투자자 보호

#### 특별 보호 조치
```
65세 이상 추정 시:
1. 큰 글씨 옵션 제공
2. 쉬운 용어 사용
3. 복잡한 상품 추가 경고
4. 사기 예방 강화 안내
```

#### 인지 부담 최소화
- 단순 명료한 설명
- 반복 설명 제공
- 시각 자료 활용
- 단계별 안내

### 청소년 보호

#### 연령 제한
```
18세 미만:
- 실제 투자 정보 제한
- 교육 콘텐츠만 제공
- 모의투자만 허용
- 부모 동의 필요
```

#### 교육 중심 접근
- 기초 금융 교육
- 리스크 이해 교육
- 장기 투자 마인드
- 도박과 투자 구분

### 초보자 보호

#### 단계별 학습 유도
```
Level 1: 기본 용어 이해
    ↓
Level 2: 투자 원칙 학습
    ↓
Level 3: 모의투자 실습
    ↓
Level 4: 소액 실전 투자
```

#### 과도한 위험 차단
- 초보자 부적합 상품 필터링
- 복잡한 전략 단계적 공개
- 필수 교육 이수 체크
- 위험도 자가 진단
