# 퀀트 투자란? (What is Quant Investing)

## 1. 개요
퀀트 투자는 **정량적 데이터(Quantitative Data)** 를 기반으로 투자 결정을 자동화하는 방식입니다.  
감이나 직관이 아니라, **수학·통계·데이터 분석**을 활용하여 투자 규칙을 만들고 백테스트를 통해 검증합니다.

---

## 2. 퀀트 투자의 핵심 특징

### ✔ 규칙 기반 투자 (Rule-Based)
- 예: “PER 낮은 상위 10개 종목 매수”
- 예: “12개월 모멘텀 상위 20% 매수”

### ✔ 감정 배제
- 공포·욕심 없이 일관된 전략 실행

### ✔ 데이터 검증
- 전략을 과거 데이터로 테스트(백테스트)
- 성과(CAGR), 위험(MDD) 분석

### ✔ 자동화 가능
- 전략 → 코드 → 자동 리밸런싱까지 확장 가능

---

## 3. 퀀트 전략의 주요 유형

### 1) 가치(Value)  
저평가된 기업에 투자  
- PER, PBR 낮음

### 2) 모멘텀(Momentum)  
상승하는 종목은 계속 오른다  
- 3/6/12개월 수익률 기준

### 3) 성장(Growth)  
고성장 기업 투자  
- 매출/EPS 성장률

### 4) 품질(Quality)  
재무 건전성 중심  
- ROE, 안정적 수익, 낮은 부채비율

### 5) 저변동(Volatility)  
하락 위험 낮은 종목 중심

---

## 4. 퀀트 투자의 장점

- 데이터 기반 → 감정∙편향 감소  
- 백테스트로 전략 검증 가능  
- 리스크 관리 용이  
- 장기적·일관적 성과  

---

## 5. 퀀트 투자의 단점

- 데이터 오류/바이어스 위험  
- 과최적화(overfitting) 주의 필요  
- 급변하는 시장에서는 과거 데이터의 무력화 가능  
- 전략 유지·관리 필요

---

## 6. 결론

퀀트 투자는  
📌 “규칙 기반 + 데이터 기반 + 자동화된 투자 방식”  
으로 초보자부터 고급 투자자까지 활용할 수 있는 강력한 방법입니다.
