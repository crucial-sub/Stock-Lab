# 가치투자 전략

## 개념

**가치투자 전략**은 저평가된 우량 기업에 투자하여 **재평가 수익을 추구하는 전략**입니다.

> 💡 **핵심**: 시장 가격 < 실제 가치 = 재평가 기회 → 수익

---

## 전략 개요

### 기본 아이디어
- 시장이 저평가한 우량 기업을 찾음
- 3~5년 이상 장기 보유
- 기업 가치가 시장에 인정될 때까지 대기
- 배당금도 함께 수익

### 투자 기간
- **권장**: 3년 이상 (장기 투자)
- **특징**: 인내심 필요, 금리 민감하지 않음

### 기대 수익률
- **연 10~15%** (보수적 추정)
- **배당 포함**: 추가 2~3%

---

## 주요 팩터

### 1️⃣ 주요 팩터 (Primary)

| 팩터 | 체크리스트 | 판단기준 |
|------|----------|--------|
| **PER** | PER < 산업평균 | 저평가 신호 |
| **PBR** | PBR < 1.0 | 자산가치 이하 |
| **배당수익률** | > 2.5% | 안정적 배당 |
| **PEG** | < 1.0 | 성장대비 저평가 |

### 2️⃣ 보조 팩터 (Secondary)

| 팩터 | 체크리스트 | 판단기준 |
|------|----------|--------|
| **ROE** | > 12% | 우량 경영 |
| **부채비율** | < 100% | 재무 안정 |
| **배당성향** | 30~50% | 지속 가능 |

---

## 실행 프로세스

### 1단계: 후보 종목 발굴

```
검색 조건:
- PER < 산업평균 × 0.8
- PBR < 1.0
- 배당수익률 > 2%
```

**예시**:
- 전자산업 평균 PER 12배 → 종목 PER < 10배
- PBR < 1.0인 종목만 선별

### 2단계: 심층 분석

```
확인 사항:
□ ROE > 10%? (우량 경영)
□ 부채비율 < 100%? (안정성)
□ 배당 성장 기록? (3년 이상)
□ 산업 전망? (긍정적인가?)
□ 경쟁력? (시장 지위)
```

### 3단계: 매수 결정

```
종합 판단:
✅ 위의 모든 항목 만족 → 매수
⚠️ 일부 미달 → 재검토
❌ 주요 항목 미달 → 제외
```

### 4단계: 장기 보유

```
보유 기간: 3~5년
배당금: 재투자 (복리 효과)
재검토: 연 1~2회 (팩터 변화 확인)
```

## 장점 ✅

| 장점 | 설명 |
|------|------|
| **안정성** | 저평가된 우량기업 → 반등 확률 높음 |
| **배당 수익** | 기다리는 동안 배당금으로 수익 창출 |
| **심리 안정** | 극단적 가격 변동 적음 |
| **세금 효율** | 배당금 정기 수익으로 예측 가능 |
| **포트폴리오** | 위험 분산에 좋은 기초 자산 |

---

## 단점 ❌

| 단점 | 설명 |
|------|------|
| **느린 수익** | 재평가까지 시간 오래 걸림 (3~5년) |
| **가치의 함정** | 저평가가 아닌 평가절하일 수 있음 |
| **경기 민감** | 경제 위기 때 추가 하락 |
| **기대치 불충족** | 예상과 달리 재평가 안 될 수 있음 |
| **금리 상승** | 금리 인상 시 배당주 매력 감소 |

## 실전 사례

### 성공 사례: 가치투자로 수익

```
구매 시점:
- 주가: 50,000원
- PER: 8배 (평균 12배)
- PBR: 0.9배
- 배당수익률: 2.5%

3년 후:
- 주가: 70,000원 (40% 상승)
- 배당금 누적: 11,250원
- 총 수익: 51,250원 (102.5%)

→ 연 25% 수익율 달성!
```

### 실패 사례: 가치의 함정

```
구매 시점:
- PER 5배 (매우 저평가)
- 왜? 실적 악화 중

3년 후:
- PER 여전히 5배
- 실적 더 악화
- 주가: 30,000원으로 하락

→ -40% 손실 (가치의 함정)
```

## 주의사항 ⚠️

### 1. 가치의 함정

**위험**: 저평가 ≠ 저가 가능성

```
예시:
- 조선업: 구조적 침체 → 영구 저평가
- 레거시 산업: 기술 대체 → 경쟁력 상실
```

**해결책**: 산업 전망, 경쟁력 철저히 분석

### 2. 재평가 타이밍

**위험**: 저평가가 몇 년 지속될 수 있음

```
시나리오:
1년: 저평가 상태 유지
2년: 여전히 저평가
3년: 아직도 저평가
→ 언제 재평가?
```

**해결책**: 촉매(Catalyst) 찾기
- 신사업 진출
- 경영진 교체
- 규제 완화
- 경기 회복

### 3. 금리 상승

**위험**: 금리 올라가면 배당주 매력 감소

```
상황:
금리: 2% → 5% 상승
→ 채권이 더 매력적
→ 배당주 회피
→ 주가 하락 가능성
```

**해결책**: 금리 트렌드 모니터링

## 체크리스트

가치주 포트폴리오 구성 전:

- [ ] 후보 15~20개 선정
- [ ] PER/PBR 모두 체크
- [ ] ROE > 10% 이상
- [ ] 부채비율 < 100%
- [ ] 배당 성장 3년 확인
- [ ] 산업 전망 긍정적
- [ ] 3년 이상 보유 가능
- [ ] 최소 5개 종목 분산 투자

## FAQ

**Q: 가치주는 항상 올라가나요?**
A: 아닙니다. 가치의 함정에 빠질 수 있습니다.

**Q: 배당금은 세금이 많이 나가나요?**
A: 네, 배당세 15.4% 부과됩니다.

**Q: 얼마나 오래 기다려야 하나요?**
A: 보통 3~5년이지만, 경우에 따라 더 오래 걸릴 수 있습니다.

**Q: 경기 침체 때는?**
A: 가치주는 경기 침체에 강한 특성이 있습니다.
