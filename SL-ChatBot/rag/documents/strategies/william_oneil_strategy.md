# 윌리엄 오닐 전략 (William O'Neil / CAN SLIM)

## 1. 개요
오닐의 CAN SLIM 전략은 성장 + 모멘텀 + 재무 건전성을 결합한  
역사상 가장 검증된 성장 모멘텀 전략 중 하나이다.

## 2. CAN SLIM 7요소
- **C (Current Earnings)**: 최근 분기 EPS 증가율 > 25%
- **A (Annual Earnings)**: 연평균 EPS 성장률 높음
- **N (New)**: 신제품, 신사업, 신고가
- **S (Supply/Demand)**: 발행주식수 적을수록 유리
- **L (Leader)**: 업종 내 리더 기업
- **I (Institutional Sponsorship)**: 기관 수급 증가
- **M (Market Direction)**: 시장 상승 추세일 때 투자

## 3. 장점
- 수십 년간 검증된 고성장 전략
- 기술적 + 펀더멘털 결합

## 4. 단점
- 조건이 엄격해 종목 수 적게 나옴
- 상승장 환경에서만 제대로 작동

## 5. 투자 성향
- 성장주 + 신고가 매매 선호
- 데이터 기반 모멘텀 투자자에게 적합
