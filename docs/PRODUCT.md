## 개요

### 서비스 소개
Stock Lab은 **퀀트 전략을 누구나 쉽게 설계·검증·운용할 수 있도록 돕는 AI 기반 투자 플랫폼**입니다.  
전문적인 퀀트 지식 없이도, AI 에이전트와의 대화를 통해 전략을 생성하고 백테스트 및 Deploy까지 한 번에 이어지는 경험을 제공합니다.

### 문제 정의
- 퀀트/팩터 전략은 진입 장벽(용어, 통계, 코드)이 높음
- 개인 투자자는 **백테스트 인프라, 데이터, 자동매매 시스템**에 접근하기 어려움
- 기존 증권사/앱의 백테스트 기능은 제한적이거나 UI가 복잡해 활용성이 낮음
- 커뮤니티 정보는 신뢰성이 낮고, 전략 성과 검증이 어려움

### 목표
- 초보 투자자: “AI가 알아서 전략을 만들어주고 요약해 주는” 경험 제공
- 경험자: “엑셀·파이썬 없이도 고급 전략을 설계·최적화·Deploy”할 수 있는 노코드 환경 제공
- 장기적으로 **전략 기반 투자 생태계(전략 생성 → 공유 → 복제 → 자동 운용)**를 구축

## 타깃 사용자 및 페르소나

### 페르소나 #1 – 투자 초보/라이트 유저
- **이름:** 김지훈, 28세, IT 스타트업 마케터
- **투자 경험:** 1~2년 (ETF/대형주 위주)
- **특징:**  
  - 용어·지표에 약함, 공부 의지는 있으나 시간 부족  
  - 데이터 기반 의사결정 욕구는 있으나, “귀찮은 건 싫음”

**목표**
- 퀀트 전략을 “쉽게” 활용하고 싶음
- 시장/종목 정보를 요약해서 받고 싶음

**Pain Points**
- 전략 용어/지표가 어렵고, 백테스트 방법을 모름
- 뉴스/시황 정보가 너무 많아 무엇을 봐야 할지 모름
- 커뮤니티 정보의 신뢰도에 의구심

**Needs**
- AI가 요약해주는 **시장·종목 브리핑**
- “선택만 하면 되는” 전략 템플릿
- 그래프/지표를 **쉬운 언어**로 풀어주는 설명
- Deploy 이후 자동 관리되는 포트폴리오

### 페르소나 #2 – 경험 많은 투자자/파워 유저
- **이름:** 박민수, 36세, 대기업 기획자
- **투자 경험:** 5~8년, 재무제표/지표 해석 가능
- **특징:**  
  - 데이터/차트에 익숙, 분석적 성향  
  - 수동 매매/엑셀 관리가 번거롭고 비효율적으로 느껴짐

**목표**
- 본인의 인사이트를 반영한 **커스텀 전략 설계**
- 충분한 백테스트와 리스크 관리 후 전략 Deploy
- 노코드 기반 자동매매 연결

**Pain Points**
- 엑셀/파이썬으로 시스템을 직접 만드는 것은 부담
- 백테스트 인프라, 데이터 클리닝, 리포트 작성에 시간 소모
- 기존 앱은 UI가 복잡하거나, 전략 설계 자유도가 낮음

**Needs**
- 모멘텀/밸류/퀄리티/변동성 등 **지표 커스터마이징**
- 샤프, MDD, 수익 곡선 등 **정교한 백테스트 리포트**
- 전략 개선 포인트를 제시하는 AI Insight
- Deploy 후 **리스크 한도 설정/리밸런싱 자동화** 옵션

## 핵심 가치 제안

1. **AI 기반 전략 자동 생성**
   - 자연어/온보딩 설문을 기반으로 맞춤 전략 템플릿을 제안
   - 초보자는 “전략 설명 + 리스크 요약” 중심의 간단한 뷰 제공

2. **노코드 퀀트 백테스트 & Deploy**
   - 엑셀·파이썬 없이도 조건/지표를 조합해 전략 설계
   - 백테스트 실행 → 성과/위험 리포트 → 실거래 Deploy까지 하나의 흐름

3. **신뢰 가능한 데이터 & 리포트**
   - 검증된 데이터 소스와 일관된 백테스트 엔진을 사용
   - 성과, 위험, 거래 내역 등을 시각화해 제공

4. **전략 중심 커뮤니티**
   - 전략 성능을 기반으로 한 랭킹/피드
   - 전략 복제, 버전 관리, 공유/피드백이 가능한 구조

## 주요 기능

### 1. HOME
- **My Portfolio Summary**
  - 전체 포트 수익률, 전략별 미니 카드, 미니 수익률 차트
- **Watchlist**
  - 관심 종목 3~5개 가격/등락률 리스트
- **AI Agent Quick Access**
  - “AI 에이전트와 대화하기” 진입 버튼
  - 오늘의 시황/포트 요약 카드

### 2. AGENT (AI 에이전트)
- **AI Chat**
  - 투자 성향, 전략 선호, 시장 질문 등 자연어 상담
  - 온보딩 질문 플로우를 통한 성향 진단
- **AI Strategy Generation**
  - 초보자: 설문 기반 전략 추천 + 자동 생성
  - 경험자: 텍스트 조건 → DSL 변환 후 전략 생성

### 3. PORTFOLIO (전략/포트 관리)
- **My Strategies**
  - 보유 전략 리스트, 성과 요약, 상태(테스트/Deploy 등)
- **Create Strategy**
  - Beginner 모드(AI Agent 주도): 템플릿 기반/자동 조건 생성
  - Advanced 모드: 지표 선택, 조건 조합, 유니버스 정의
- **Backtest Report**
  - 누적 수익률, MDD, Sharpe, 월별 성과, 트레이드 로그, 벤치마크 비교
- **Deploy**
  - 브로커리지(예: 키움) 연동, 포트 구성 자동 제안
  - 리스크 한도/리밸런싱 조건 설정

### 4. MARKET
- **Market Overview**
  - 주요 지수/섹터/테마 요약
  - AI 요약 시황 코멘트
- **Stock Finder**
  - 팩터/지표 기반 종목 스크리닝
  - 전략과 연계 가능한 종목 후보 탐색

### 5. News
- **Market News**
  - 전체 시장/섹터 주요 뉴스 피드
- **Stock News + AI Summary**
  - 종목별 뉴스 묶음 + AI 요약
  - 내 포트/전략 관련 이슈 우선 정렬

### 6. COMMUNITY
- **Strategy Feed**
  - 인기/신규/성과 기반 전략 피드
- **Strategy Sharing**
  - 자신의 전략을 공개/비공개/공유 범위 선택
- **AGORA**
  - 전략 토론, 피드백, 인사이트 공유 게시판

### 7. MY PAGE
- **Profile**
  - 사용자 정보, 투자 성향, 선호 전략 타입
- **Brokerage Linked**
  - 증권사 계좌 연동 현황, 토큰 만료/갱신 관리
- **Help Center**
  - 튜토리얼, FAQ, 고객 지원 채널

## 사용자 여정(User Journey)

### 투자 초보 (Persona #1)
1. 앱 실행 → 온보딩 시작  
2. “AI가 전략을 만들어드립니다” 튜토리얼 확인  
3. 3~5문항 투자 성향 설문  
4. “당신에게 맞는 전략 찾기” → AI가 2~3개 전략 추천  
5. 전략 상세 화면에서 쉬운 설명/핵심 지표/리스크 요약 확인  
6. “전략 만들어보기” 클릭 → 자동 전략 구성 + 자동 백테스트 실행  
7. 요약 카드(수익률/위험도/간단 인사이트) 확인  
8. “Deploy 하기” → 포트 구성 자동 제안 → 확인 후 Deploy 완료  
9. 홈 대시보드에서 시황 요약 + 내 포트 성과를 주기적으로 확인  

### 경험 많은 투자자 (Persona #2)
1. 앱/웹 실행 → 간단 온보딩(스킵 가능)  
2. 전략 탐색 페이지로 바로 진입  
3. 인기 전략 탐색 / 지표 검색 / “커스텀 전략 만들기” 중 선택  
4. 커스텀 전략 생성 시, 모멘텀/밸류/퀄리티/변동성/사용자 정의 조건 선택  
5. 지표 조건 조합 + 종목 유니버스 설정  
6. 필요 시 “AI에게 전략 최적화 요청” → 파라미터 추천  
7. 백테스트 실행 → 상세 리포트(성과, 위험, 트레이드 로그, 벤치마크 비교) 확인  
8. 전략 저장 + 버전명 부여 → 기존 전략/벤치마크와 비교  
9. Deploy 시 리스크 한도/리밸런싱 조건 설정 → 포트 생성  
10. 대시보드에서 성과를 2~3일마다 확인, 성과 저하 시 전략 버전 업데이트 및 재백테스트

## UX 원칙

1. **전문 용어 최소화**
   - 초보 유저 화면에서는 지표/팩터를 최대한 Plain Language로 설명
   - 상세 전문 정보는 “더보기/자세히 보기”로 분리

2. **결과 요약 → 상세 설명 구조**
   - 홈/백테스트/전략 상세 등 모든 주요 화면에서 상단은 요약, 하단은 상세
   - 초보자는 요약만으로도 의사결정이 가능하도록 설계

3. **질문형 온보딩**
   - “어떤 투자 스타일인가요?”, “손실을 어느 정도까지 감내할 수 있나요?” 등 질문 기반 플로우
   - 응답을 기반으로 전략 추천/리스크 한도 기본값 설정

4. **AI 주도 인터페이스**
   - 사용자가 주저할 만한 지점에 AI가 먼저 제안(예: “이 전략을 최적화해볼까요?”)
   - 전략 성과 변화에 대한 자연어 브리핑 제공

5. **Progressive Disclosure**
   - 고급 기능(지표 커스터마이징, 트레이드 로그, 상세 리스크 분석)은 기본 뷰에서 숨김
   - 필요 시 확장/드롭다운으로 단계별 공개

## 향후 확장 및 로드맵

1. **멀티 브로커리지 연동**
   - 초기 키움증권 중심 → 타 증권사/글로벌 브로커로 확장
2. **전략 마켓플레이스**
   - 유료/무료 전략 구독, 수익 공유 모델 도입
3. **고급 분석 모듈**
   - 팩터 기여도 분석, 슬리피지/거래비용 모델링, 리밸런싱 시뮬레이션
4. **모바일 퍼스트 최적화**
   - 모바일 앱 정식 출시, 푸시 알림 기반 리밸런싱/리스크 알림 강화
5. **AI 에이전트 고도화**
   - 개인화 메모리(사용자 과거 응답/전략) 기반 초개인화 상담
   - 음성 인터페이스, 멀티모달 차트 설명 등 지원
